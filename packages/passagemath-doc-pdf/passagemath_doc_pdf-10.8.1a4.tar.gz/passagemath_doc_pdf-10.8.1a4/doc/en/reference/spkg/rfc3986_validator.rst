       $ pip install rfc3986-validator

   .. CODE-BLOCK:: bash

       $ sudo pacman -S ${MINGW_PACKAGE_PREFIX}-python-rfc3986-validator

``--enable-system-site-packages`` is passed to ``./configure``, then
