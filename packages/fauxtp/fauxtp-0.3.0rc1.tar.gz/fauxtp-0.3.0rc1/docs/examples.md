# Examples

The `fauxtp` repository includes several examples demonstrating how to use the library for various concurrent patterns.

## [Concurrent KV Store](https://github.com/fizzAI/fauxtp/blob/main/examples/01_concurrent_kvstore.py)
A simple key-value store implemented as a `GenServer`.

## [Pipeline Processing](https://github.com/fizzAI/fauxtp/blob/main/examples/02_pipeline_processing.py)
Demonstrates how to build a processing pipeline using multiple actors.

## [Task Queue](https://github.com/fizzAI/fauxtp/blob/main/examples/03_task_queue.py)
A basic task queue implementation.

## [PubSub System](https://github.com/fizzAI/fauxtp/blob/main/examples/04_pubsub_system.py)
A simple publish-subscribe system using actors.

## [Stack](https://github.com/fizzAI/fauxtp/blob/main/examples/human/stack.py)
A human-readable example of a stack implemented with `GenServer`.
