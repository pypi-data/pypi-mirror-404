# 颌痛

关键问题 姓名，年龄，起始时间，过敏史，用药情况，既往病史，疼痛放射部位，伴随症状，疼痛评分

需考虑的其他方案 胸痛（123）；胃酸倒流（316）；消化不良（368）；口腔问题（407）；颈部疼痛（415）；牙痛（613）；牙齿损伤（616）。

护士提示：引起颌痛的病症很多，其中一些可能是潜在的生命威胁。在处理报告颌痛的来电者时，要倾向于谨慎行事。出现颌、颈或背部疼痛、恶心和/或呼吸急促症状的女性可能正在经历心脏病发作。

提醒：记录来电者对建议的反应，家庭护理指导以及何时回电。

## 评估与行动

A. 是否存在以下任何情况？

●胸痛

●呼吸困难

是 转至胸痛方案（123）或呼吸问题方案（106）

否 ↓ 转至B

B. 是否存在以下任何情况？

●间歇性颌痛，有高血压、心血管疾病、高胆固醇、肥胖症、重度吸烟史或年龄>30岁病史

●新发疼痛并向颈部、肩部或手臂放射

是 “立即寻求紧急医疗”

否 ↓ 转至C

C. 是否存在以下任何情况？

●剧烈疼痛

●近期该区域受到创伤

●牙齿不再像往常那样对齐

是 “2到4小时内寻求医疗或牙科护理”

否 ↓ 转至D

D. 是否存在以下任何情况？

●颌部在某些位置锁定

●感染迹象（增加的疼痛、肿胀、分泌物、红色条纹或温暖感）

是 “24小时内寻求医疗或牙科护理”

否 ↓ 转至E

E. 是否存在以下任何情况？

●持续时间>2周的疼痛

●头痛，颈部和肩部疼痛

●颌动时有咔嚓声、啪嗒声或爆裂声

●张口困难

●颞下颌关节痛

●耳痛或眼痛

●该区域间歇性肿胀

是 “如果症状没有改善，请回电或联系家庭医生/牙医预约”并遵循家庭护理指导

否 遵循家庭护理指导



## 家庭护理指导

颌痛

●避免咀嚼口香糖或硬食物。

●每天交替使用冷敷和热敷6次，每次应用于颌部。

●避免将电话听筒夹在颌与肩之间。

●根据耐受情况服用常规止痛药（阿司匹林、对乙酰氨基酚、布洛芬）以缓解不适和肿胀。不要给儿童服阿司匹林。20岁以下者避免使用类似阿司匹林的产品。如果存在肝脏疾病，则避免使用对乙酰氨基酚。如果存在肾脏疾病或胃部问题，或者在怀孕期间，请勿使用布洛芬。请按照标签上的指示服用。

●按摩颌周围的肌肉。

## 如有以下情况，请向您的家庭医生/牙医/诊所报告

●经家庭护理措施后持续性疼痛未缓解

●病情恶化

●感染迹象：增加的疼痛、肿胀、分泌物、红色条纹、温暖感或发热



## 如果出现以下任何情况，请立即寻求紧急医疗

●胸痛

●呼吸困难

●新发疼痛并向颈部、肩部或手臂放射

如果来电者同意所给建议，记录通话并鼓励其在问题恶化时回电或就诊于家庭医生。

如果来电者不同意所给建议，重新评估并建议其向家庭医生、诊所或急诊科复诊。
