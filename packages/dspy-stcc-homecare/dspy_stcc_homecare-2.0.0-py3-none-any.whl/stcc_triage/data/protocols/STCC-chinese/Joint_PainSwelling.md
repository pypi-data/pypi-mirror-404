## 评估与行动

A. 下列情况是否出现？

●踝部肿胀伴有胸痛、咳血或呼吸困难

●脱位或畸形

●受影响部位的手指或脚趾变冷或发绀

是 “呼叫救护车” 或 “立即寻求紧急医疗帮助”

否 ↓ 继续B

B. 下列情况是否出现？

●大腿或小腿的肿胀和疼痛

●新出现且无法行走或承重

●关节、小腿或大腿在没有已知损伤的情况下疼痛、肿胀、发热或发红

●手臂和/或腿上有紫色皮疹的儿童

●一侧肢体肿胀，近期有长途旅行史、怀孕、癌症病史、长期卧床不起或腿部血栓史

是 “2到4小时内寻求医疗帮助”

否 ↓ 继续C

C. 下列情况是否出现？

●孕期且症状新发

●疼痛持续或加重于：

●行走或站立时

●休息时

●抬腿时

●脚部屈曲时

●使用手、臂或肩部时

●有心脏病、肝病、肾病、免疫抑制、糖尿病或近期疾病史，如喉咙痛或皮肤感染

●新处方药物

●关节或大拇指根部剧烈疼痛

●关节上方的皮肤发红或光亮

●膝盖或髋部疼痛且跛行的儿童

●具有以下两种情况：头痛、喉咙痛、咳嗽

●不使用手臂或手的儿童

●发热

●近期体重增加超过10磅（4.5公斤）

是 “24小时内寻求医疗帮助”

否 ↓ 继续D

D. 下列情况是否出现？

●孕期

●其他关节疼痛

●全身不适感

●突然发作且无已知损伤

●轻微肿胀

●家庭护理措施无法缓解的慢性疼痛

是 “如果症状没有改善，请回电或联系初级保健医生（PCP）预约” 并遵循家庭护理指示

否 遵循家庭护理指示



## 家庭护理指示

关节疼痛/肿胀

●抬高受影响肢体，使其高于心脏。在小腿下垫枕头以减轻踝部肿胀。

●对于踝部肿胀，减少饮食中的盐分摄入。

●如果没有已知损伤，可对患处进行热敷：使用时需小心，尤其是糖尿病患者不要睡着时仍然放置加热垫上。

●不要按摩疼痛的大腿或小腿。

●服用常规止痛药（阿司匹林、扑热息痛、布洛芬）。不要给儿童服用阿司匹林。年龄小于20岁者避免使用类似阿司匹林的药物。肝病患者避免使用扑热息痛。肾病或胃部问题存在时，或孕期时避免使用布洛芬。遵循标签上的说明。如果正在接受抗凝治疗，在服用阿司匹林前请咨询初级保健医生（PCP）。

●休息。

●对于已知损伤，关节处敷冰袋20至30分钟，每2小时一次，持续24至48小时内。不要将冰直接放在皮肤上；在冰和皮肤之间放置一条湿布或其他织物隔开。

## 如果出现以下问题，请向初级保健医生/诊所/急诊报告

●症状没有改善或加重

●疼痛加剧

●活动能力下降

●发热



## 如果出现下列任何情况，应立即寻求紧急医疗帮助

●踝部肿胀伴有胸痛、咳血或呼吸困难

●受影响部位的手指或脚趾变冷或发绀

如果来电者同意给予的建议，请记录通话并鼓励在问题恶化时回电或咨询初级保健医生（PCP）。

如果来电者不同意所给的建议，重新评估，并建议与初级保健医生、诊所或急诊联系。
