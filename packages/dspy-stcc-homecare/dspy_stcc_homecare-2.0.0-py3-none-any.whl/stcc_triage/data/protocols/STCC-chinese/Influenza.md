# 流感

关键问题：姓名、年龄、发病时间、症状、已知接触史或社区疫情、用药情况、病史

其他考虑的方案：禽流感（55）；普通感冒症状（146）；鼻塞（153）；咳嗽（170）；成人体温升高（250），儿童体温升高（253）；头痛（308）；猪流感（H1N1病毒）暴露（600）；咽痛（564）；西尼罗河病毒（653）。

护士提醒：如果已知或怀疑有流感接触史，社区出现疫情，或者之前已被诊断为流感，请使用此方案。

提示：记录来电者对建议的反应、家庭护理指导以及何时回电。

## 家庭护理指导

流感

●经常用肥皂和水洗手，或使用酒精基手部消毒剂。

●强调流感具有高度传染性。保持良好的呼吸道礼仪；咳嗽或打喷嚏时用纸巾遮住口鼻。

●避免与生病的人接触。

●如果感到不适，避免与其他人接触。咳嗽和打喷嚏时，在与人近距离接触时佩戴外科口罩以防止飞沫传播。如果口罩变脏或湿润，请更换。

●充分休息并多喝水。

●不要给儿童服用阿司匹林。年龄<20岁的人应避免使用类似阿司匹林的产品。肝病患者避免使用对乙酰氨基酚。存在肾病、胃部问题或怀孕的情况下，避免使用布洛芬。请遵循标签上的指示。

●在症状出现后的48小时内给予抗病毒药物。
