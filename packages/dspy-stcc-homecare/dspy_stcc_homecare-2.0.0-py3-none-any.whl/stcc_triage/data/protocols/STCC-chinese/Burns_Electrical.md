# 烧伤，电击

关键问题 姓名、年龄、发病时间、原因、伤口入口和出口位置、破伤风免疫状态、用药情况、疼痛评分、病史

其他考虑的协议 意识障碍 (28)；烧伤，热力 (118)；电击伤害 (219)；抽搐 (531)；伤口愈合与感染 (664)。

提醒：记录来电者对建议的反应、家庭护理指导以及何时回拨。

## 评估和行动

A. 是否存在以下任何情况？

●高电压电击

●意识丧失或无脉搏或呼吸

●胸痛

●心跳加速或不规则

●呼吸困难

●颈部以上烧伤

●明显的入口和出口伤口

●闪电引起的烧伤

是 “呼叫救护车”

并

“如果无脉搏或呼吸，开始进行心肺复苏或人工呼吸”

否 ↓ 转到 B

B. 是否存在以下任何情况？

●由电弧或闪击造成的烧伤

●头发被烧焦或缺失但皮肤没有烧伤

●口腔或唇部的烧伤，尤其是婴儿或幼儿

●环绕颈部或肢体的烧伤

●关节上的烧伤

●失忆或有任何意识丧失期

●从电源弹开并伴有呼吸困难、胸痛或腹痛

●麻木、刺痛或瘫痪；视力、听力或言语问题

●头部、面部、颈部、手部、脚部或生殖器区域的烧伤

是 “立即寻求紧急医疗”

否 ↓ 转到 C

C. 是否存在以下任何情况？

●低电压小范围烧伤

●怀孕超过20周

是 “立即就医”

否 ↓ 转到 D

D. 是否存在以下任何情况？

●没有破伤风免疫接种，或免疫状态未知，或最后一次破伤风免疫接种超过5年前

●烧伤愈合不良

●感染迹象：红肿、条纹状红斑、肿胀或有分泌物

●没有其他症状，但个人或家长担忧

是 “24小时内就医”

否 遵循家庭护理指导


## 家庭护理指导

电击烧伤

●停止燃烧过程。脱下着火的衣物和首饰。

●用干净湿润的冷布冷却烧伤区域。

●不要涂抹油膏、油脂、黄油或止痛药膏。

●保持伤口清洁，用干净干燥的无粘性敷料覆盖。

●观察感染迹象。

●服用常规使用的止痛药。儿童不得使用阿司匹林。<20岁的患者避免使用类似阿司匹林的产品；肝病患者避免使用对乙酰氨基酚；肾病或胃部问题的患者以及孕妇不要使用布洛芬。遵循标签上的指示。

●不要刺破或排空水泡。

●如果电击烧伤是由电器引起的，在修复前不得使用该电器。在浴室、厨房和室外安装接地故障断路器（GFCI）。如果没有GFCI插座，可使用带GFCI功能的延长线。

## 向您的PCP/诊所/急诊科报告以下问题

●烧伤愈合不良

●感染迹象：红肿、条纹状红斑、肿胀或有分泌物

●肌肉疼痛或无力

●闪电打击后的头痛、耳痛或记忆丧失

●病情恶化或无改善



## 如果出现下列任何情况，立即寻求紧急医疗

●抽搐

●呼吸困难

●胸痛

●麻木、刺痛或瘫痪；视力、听力或言语问题

如果来电者同意给出的建议，请记录通话，并鼓励来电者在问题恶化时回拨或咨询PCP。

如果来电者不同意给出的建议，请重新评估并建议来电者联系PCP、诊所或急诊科。
