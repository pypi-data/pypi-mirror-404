## 家庭护理指导

口臭

●使用抗菌漱口水（如李施德林）漱口。

●每天刷牙、假牙、牙龈和舌头两次，并定期使用牙线。

●每日取下并浸泡桥冠和假牙。

●如果认为维生素或矿物质补充剂是导致口臭的原因，减少其剂量。

●减少使用铸铁炊具（除非医生建议）。

●减少食用大蒜、洋葱、酒精或咖啡。

●定期看牙医，并在出现牙齿问题时及时就诊。
