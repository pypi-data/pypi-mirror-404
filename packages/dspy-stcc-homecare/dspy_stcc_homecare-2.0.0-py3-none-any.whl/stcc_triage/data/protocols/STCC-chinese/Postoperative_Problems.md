# 术后问题

关键问题 姓名、年龄、发病时间、原因、手术类型及日期、能否联系到外科医生、石膏/夹板/敷料的位置、药物使用情况、病史、疼痛评分

其他需要考虑的协议 出血，严重（90）；呼吸问题（106）；石膏/夹板问题（121）；胸痛（123）；便秘（160）；发热，成人（250），儿童（253）；腿痛/肿胀（398）；呕吐，成人（642），儿童（645）；肿胀（597）；无力（649）；伤口护理：缝线或钉子（660）；伤口愈合和感染（664）。

提醒：记录来电者对建议、家庭护理指导的反应，以及何时需要回电。

## 评估与行动

A. 下列任何情况是否出现？

●突然出现剧烈疼痛

●手术伤口裂开或张口，并有大量液体流出或物体从伤口突出

●如果是石膏、夹板或骨科手术：

●手指或脚趾冷、蓝或麻木

●严重疼痛、肿胀或紧绷，对抬高或家庭护理措施无反应

●胸痛

●咳血或粉红色泡沫痰

●腿部肿胀和受影响的腿脚脉搏消失、麻木或刺痛

●呼吸短促

是 “叫救护车” 或 “立即寻求紧急医疗”

否 ↓ 转到B

B. 下列任何情况是否出现？

●手术伤口裂开或张口

●感染迹象（疼痛、肿胀、流液、发热或红条从伤口延伸）

●大腿、小腿、踝部或脚趾的肿胀和疼痛

●敷料、夹板或石膏感觉太紧

●切口大量出血，对家庭护理措施无反应

●脚踝、小腿、胫骨或大腿区域触感温暖或发红

●一条腿或脚踝突然肿胀

●体温 >100.4°F (38°C)

是 “数小时内寻求医疗”

否 ↓ 转到C

C. 下列任何情况是否出现？

●糖尿病、HIV感染、慢性病史，使用类固醇、抗凝血药或化疗药物且伤口愈合不良

●疼痛逐渐加剧

●服用处方止痛药后疼痛未缓解

●石膏、夹板或器具开裂或不稳定

●切口处持续渗出或出血

●持续恶心和呕吐

是 “24小时内寻求医疗”

否 ↓ 转到D

D. 下列任何情况是否出现？

●伤口边缘瘙痒

●在石膏或夹板内有持续的肿胀或紧绷，但经家庭护理措施后有所改善

●敷料上有少量血性渗出物

●便秘

●有关活动或饮食的问题

●负压伤口治疗（VAC）装置就位且

●漏气

●容器已满

●关闭8小时以上

是 “如果无好转，请回电或预约家庭医生”并遵循家庭护理指导

否 遵循家庭护理指导



## 家庭护理指导

术后问题

●定时服用止痛药。

●按照家庭医生的指示执行家庭护理计划，包括抗生素、伤口检查、运动、饮食、肠道护理和抬高肢体。

●根据家庭医生的指示保持伤口清洁干燥。

●记住，伤口边缘瘙痒是愈合的迹象。

●如果负压伤口治疗（VAC）装置使用中出现任何问题，请立即联系医疗提供者。

●胸痛

●咳血或粉红色泡沫痰
