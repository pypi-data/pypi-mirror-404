# 新生儿问题

关键问题：姓名、年龄、起始时间、原因、出生史、用药情况

其他可考虑的协议：母乳喂养问题（98）；包皮环切术后护理（141）；婴儿过度哭泣（176）；儿童发热（253）；黄疸（385）；婴儿睡眠呼吸暂停（562）。

护士提示：新生儿可能受多种条件影响。当出现多重相关症状时，应使用主要关注点且最有可能需要转诊至更高级别护理的协议。询问婴儿当前状况与平时模式有何不同，以及在打电话时婴儿的行为如何。

提醒：记录来电者对建议、家庭护理指示和何时回电的反应。

## 评估，行动

A. 是否存在以下任何情况？

●抽搐

●无反应

●直肠温度 >100.4°F (38°C)

●体温 <96.8°F (36°C) 且对保暖无反应

●囟门凸起伴有呕吐、发热或行为异常

●跌落硬表面并有明显受伤

●不褪色的皮疹

●皮疹或瘀伤

是 “呼叫救护车” 或 “立即前往急诊科”

否 ↓ 继续 B

B. 是否存在以下任何情况？

●跌落硬表面但无明显受伤

●拒绝饮水 >6 小时

●脐带出血超过 10 分钟的直接压迫

●脐周有红色条纹或红斑区域

●身体某部位有一簇水泡

●水痘

●呼吸急促

●呼吸困难并伴有喘息声

●难以安抚

●鼻翼扇动

●皮肤、嘴唇或指甲床呈苍白或蓝色

●腰部以下、手臂或腿部有黄色色调

●皮肤发黄且 <24 小时大

是 “立即寻求医疗护理”

否 ↓ 继续 C

C. 是否存在以下任何情况？

●囟门凸起但婴儿行为正常

●脐带间歇性出血 >3 天

●接近脐带的丘疹或水泡

●中等量脐部引流但无发热

●眼睛有黄色粘液分泌物

●红眼或肿胀的眼睑

●出生超过 7 天且皮肤发黄

●包皮环切术后有黄色或绿色阴茎分泌物（无发热）

●>24 小时未排便

●白色、黄色或陶土色的大便

是 “24 小时内寻求医疗护理”

否 ↓ 继续 D

D. 是否存在以下任何情况？

●间歇性囟门凸起但无疾病迹象或症状

●脐带少量出血

●肚脐内的粉红色组织

●>2 天的脐带有恶臭（无发热或红斑）

●呼吸时有喘息声但无呼吸困难

是 “如果状况没有改善，请回电或预约家庭医生” 并遵循家庭护理指示

否 遵循家庭护理指示



## 家庭护理指示

新生儿问题

提供安慰：

●出生后第一周内乳房肿胀是正常的。

●眼球白色部分的红色条纹与分娩过程有关，将在 2 到 3 周内消退。

●水汪汪的眼睛通常是由于泪管堵塞所致，一年内会自行解决。注意观察感染迹象，如红、肿或有分泌物。

●阴道出现清澈、白色或粉红色分泌物是正常的，并会在 2 到 3 天内消失。

●阴囊肿胀将在 6 至 12 个月内消退。

●唤醒困倦新生儿的小贴士：

●解开毯子并脱去婴儿的衣服。

●更换尿布。

●按摩婴儿的腿、背和手臂。

●通过用手指沿着脊椎轻拍给婴儿背部按摩。

●进行“仰卧起坐”：将婴儿远离自己，轻轻将婴儿拉向自己。

其他注意事项参见相关协议。

## 如果出现以下任何情况，请立即寻求紧急护理

●抽搐

●无反应

●体温 >100.4°F (38°C)

●囟门凸起伴有呕吐、发热或行为异常的新生儿

●暗红色或紫色不褪色的皮疹

如果来电者同意所给建议，记录通话并鼓励其在问题恶化时回电或看家庭医生。

如果来电者不同意所给建议，请重新评估，并建议其联系家庭医生、诊所或急诊科。
