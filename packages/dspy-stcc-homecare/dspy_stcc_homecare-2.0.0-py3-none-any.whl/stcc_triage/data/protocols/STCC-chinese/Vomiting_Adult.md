## 家庭护理指导

成人呕吐

●最后一次呕吐后1小时内不要吃任何东西或喝水。

●最初12小时内，每次只喝少量的清澈液体，包括明胶、水、运动饮料、无气泡的汽水、清汤、冰块（带味）或苹果汁。不要喝柑橘类果汁。

●根据耐受情况逐渐增加液体摄入量。

●12小时后尝试吃小份的清淡食物，如米饭、土豆、苏打饼干、椒盐脆饼、干面包和苹果酱。

●在能够耐受清淡食物后，逐步恢复正常饮食，根据耐受情况进行调整。

●如果服用常规药物30分钟内出现呕吐，请重新服药。

●避免牛奶、柑橘类食品和果汁、辛辣和高脂肪的食物、酒精、咖啡或含咖啡因的饮料。

●可以服用非处方的美沙酮（如 Dramamine）或磷酸碳水化合物溶液（如 Emetrol）来缓解恶心，按照包装上的说明使用。

●引起恶心和呕吐的病毒很容易传播。特别注意手部卫生。避免使用感染者用过的毛巾、餐具和杯子。
