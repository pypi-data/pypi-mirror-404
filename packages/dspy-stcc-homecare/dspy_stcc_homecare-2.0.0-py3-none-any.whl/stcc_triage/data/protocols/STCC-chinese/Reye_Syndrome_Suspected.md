# 瑞氏综合征，疑似

关键问题：姓名、年龄、发病时间、用药情况、病史

其他需考虑的协议：水痘（129）；意识混乱（150）；充血（153）；咳嗽（170）；儿童发热（253）。

护士提示：如果之前被诊断为瑞氏综合征，或者呼叫者怀疑在近期疾病后服用阿司匹林的儿童或青少年现在出现了新的令人担忧的症状，请使用此协议。

提醒：记录呼叫者对建议、家庭护理指导及何时回电的回应。

## 评估与行动

A. 是否存在以下任何情况？

●最近有病毒性疾病史，如水痘、流感或其他呼吸道疾病

●在疾病期间服用阿司匹林，并且在病毒感染后不久出现以下任何症状：

●头痛、协调障碍、言语不清、意识混乱、倦怠、性格变化

●呕吐

●肢体无力

●听力下降或复视

存在 “立即寻求急救”

不存在 “若无改善，请回电或预约PCP” 并遵循家庭护理指导



## 家庭护理指导

瑞氏综合征，疑似

●通常不建议给儿童或青少年使用阿司匹林，因为症状往往在病毒感染后的几天才出现。对于发热和不适可服用对乙酰氨基酚。

●如果孩子在病毒性疾病期间或之前立即服用了阿司匹林，请密切观察是否有瑞氏综合征的症状，并报告给您的PCP。这些症状通常出现在病毒性疾病的急性期之后，可能迅速进展。包括以下症状：

●头痛、协调障碍、言语不清、意识混乱、倦怠和性格变化

●呕吐

●肢体无力

●听力下降或复视

额外指导：



向您的PCP/诊所/急诊科报告以下问题：

●疑似瑞氏综合征的症状



## 如果出现以下任何情况，请立即寻求急救

●意识混乱、倦怠和其他精神状态改变

●头痛、协调障碍、言语不清

●呕吐

●肢体无力

●听力下降或复视

如果呼叫者同意所给的建议，记录通话并鼓励呼叫者在问题恶化时回电或见PCP。

如果呼叫者不同意所给的建议，重新评估，并建议呼叫者联系PCP、诊所或急诊科。
