# 混乱

关键问题：姓名、年龄、起始时间、原因、用药、病史、疼痛评分

其他需考虑的协议：酒精问题（21）；意识状态改变（28）；成人发热（250），儿童发热（253）；头痛（308）；电话分诊中的心理健康挑战（App V 720）；癫痫发作（531）；热性惊厥（533）；物质滥用、使用或暴露（5）。

护士提醒：混乱的迹象可能包括易怒；对声音或触摸反应减弱；嗜睡；好斗，不合作，言语无意义；行为、思维过程或沟通能力突然改变；听觉幻觉（听到声音、嗡嗡声、咔嗒声），感觉幻觉（虫子爬行感），视觉幻觉。

●在老年人中，混乱可能是尿路感染、脱水或中风的最早迹象之一。

●在儿童中，混乱可能是快速进展的脑膜炎或创伤后头部损伤的最早迹象之一。

提醒：记录呼叫者对建议的反应、家庭护理指导以及何时回拨。

## 家庭护理指导

混乱

●如果正在服用可能导致谵妄的药物（如抗组胺药、颠茄碱、生物碱），请停止使用，并在1小时内如果没有改善，请回拨或联系家庭医生。

●让患者在一个光线充足、熟悉的环境中舒适地休息，旁边有人陪伴。

●给予常规退烧药（如对乙酰氨基酚）。用温水浴降低体温。不要给儿童服用阿司匹林。20岁以下者避免使用类似阿司匹林的产品。肝病存在时避免使用对乙酰氨基酚。肾病、胃部问题或怀孕的情况下避免使用布洛芬。请按照标签上的说明使用。
