## 家庭护理指导

昏厥

●对于头昏眼花，可将双腿抬高过头部，或坐下并将头部置于两膝之间直至不适感消失。

●若长时间暴露于热环境中，应小口饮用凉饮料，并使用冷敷物以帮助降温。

●避免突然改变姿势：从躺着、坐着或弯腰的姿势缓慢起身。

●如果病人患有糖尿病，请检查血糖并采取适当措施。对于低血糖，可喝一杯橙汁、可乐或牛奶。

●频繁食用少量蛋白质零食。如果超重，则应遵循均衡、合理的减重饮食计划。

●避免长时间站立不动。可能的话，变换双脚的重量分布，走动一下。
