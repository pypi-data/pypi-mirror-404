# 喉炎

关键问题：姓名、年龄、发病时间、咳嗽描述、先前治疗、用药情况、伴随症状、病史

其他需要考虑的协议：呼吸困难（106）；充血（153）；咳嗽（170）；成人发热（250），儿童发热（253）；流感（372）；百日咳（441）；咽喉痛（564）。

提醒：记录来电者对建议的反应、家庭护理指导以及何时回电。

## 评估与行动

A. 是否存在以下任何情况？

●流口水，吞咽困难，孩子看起来病态

●嘴唇发绀或呈灰暗色

●严重呼吸困难

●呼吸时胸部凹陷

是 “呼叫救护车”

否 ↓ 转至B

B. 是否存在以下任何情况？

●吸气时有鸡鸣样声音，经过20分钟的蒸汽处理后仍未缓解，或者在过去24小时内出现三次以上此类症状

●孩子看起来非常病态

是 “立即寻求紧急护理”

否 ↓ 转至C

C. 是否存在以下任何情况？

●肺炎或其他肺部疾病史

●哮喘史且在家使用呼吸治疗和常规支气管扩张剂后无改善

是 “2小时内就医”

否 ↓ 转至D

D. 是否存在以下任何情况？

●躺下时病情加重

●影响睡眠

●体温 >104°F (40°C)

是 “24小时内就医”

否 ↓ 转至E

E. 是否存在以下任何情况？

●犬吠样咳嗽超过5天

●夜间咳嗽加剧

●白天听到犬吠样咳嗽

是 “如果无改善，请回电或预约主治医生”

并

遵循家庭护理指导

否 遵循家庭护理指导



## 家庭护理指导

喉炎

●温湿空气可能有助于病情：通过覆盖在口鼻上的温暖湿润的毛巾呼吸，或者坐在充满蒸汽的浴室中20到30分钟。

●持续的冷雾可能有所帮助。使用冷雾加湿器或外出进入凉爽的夜晚空气中。

●喝温热透明液体以缓解咳嗽。

●按照常规剂量给发热的孩子服用药物（儿童美林、布洛芬、儿童泰诺、对乙酰氨基酚）。不要给孩子服用阿司匹林。如果年龄小于20岁，避免使用类似阿司匹林的产品。如果有肝病，避免使用对乙酰氨基酚。如果有肾病或胃部问题存在，或者在怀孕的情况下，避免使用布洛芬。请遵循标签上的指示。

## 出现以下任何问题时请联系您的主治医生/诊所/急诊室

●无改善或病情加重

●呼吸困难

●体温 >104°F (40°C)

●哮喘儿童在家进行呼吸治疗后病情持续存在



## 如果出现以下任何情况，立即寻求紧急护理

●流口水、吞咽困难且孩子看起来病态

●嘴唇变蓝或发暗

●严重呼吸困难

●孩子看起来非常病态

●呼吸时胸部凹陷

●经过20分钟的家庭护理措施后鸡鸣样声音未缓解，或者在24小时内出现三次以上此类症状

如果来电者同意所给建议，请记录通话并鼓励他们在问题加重时回电或就诊于主治医生。

如果来电者不同意所给建议，请重新评估，并建议他们与主治医生、诊所或急诊室联系。
