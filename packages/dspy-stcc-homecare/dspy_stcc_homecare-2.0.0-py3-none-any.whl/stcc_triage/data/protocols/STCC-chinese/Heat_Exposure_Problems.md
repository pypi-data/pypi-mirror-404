# 热暴露问题

关键问题：姓名、年龄、发病时间、原因、体温、药物、病史

其他需考虑的方案：意识状态改变（28）；脱水（180）；头晕（199）；昏厥（237）；成人发热（250）、儿童发热（253）；肌肉痉挛（413）；晒伤（588）；多汗症（594）；虚弱（649）

护士提醒：

●高热可能由长时间暴露于高温或潮湿环境中、过度运动、感染、使用如安非他命等药物以及老年人散热能力受损引起。

●暴露在非常温暖环境中的老年人更容易脱水和泌尿系统感染，意识状态改变可能是需要及时关注的问题的首个迹象。

提醒：记录呼叫者对建议的反应、家庭护理指导及何时再次致电。

## 家庭护理指导

热暴露问题

紧急家庭护理指导

●帮助受害者躺下在阴凉的地方，松开衣服或脱掉衣物。

●在等待医疗援助时保持受害者冷静。使用冷毛巾并用风扇帮助降温。不要直接将冰放在皮肤上。不要让受害者发抖，因为这会增加体温。用温水擦拭受害者。不要使用酒精擦洗。

●抬高双腿至高于心脏的位置。

●如果意识清醒，给受害者喝冷水（水、运动饮料、软饮料、果汁）。不要饮用含酒精或咖啡因的饮料。过快饮水会导致恶心。每15分钟喝一杯，如能耐受且无恶心感可继续。

中度或轻度热暴露指导

●使用喷瓶或喷雾器喷温水到暴露的皮肤上，通过蒸发帮助降温。

●如果年龄 <13 岁，给予佩迪莱特或赖莱特以补充水分。

●在一夸脱水中加入 ¼ 茶匙盐。每15分钟给予 ½ 杯，持续 1.5 小时。不要让患者服用盐片。如果口渴持续，请给普通水。

●如果没有恶心感，可以尝试吃一些咸味小吃，如椒盐脆饼或薯条。

## 如果出现以下任何情况立即寻求紧急医疗帮助

●警觉性下降

●抽搐

●肌肉痉挛或不协调

●持续异常缓慢的心率

●持续呕吐和无法耐受液体

如果呼叫者同意所给的建议，记录通话并鼓励呼叫者在问题恶化时回电或见家庭医生。

如果呼叫者不同意所给的建议，重新评估，并建议呼叫者联系家庭医生、诊所或急诊室。
