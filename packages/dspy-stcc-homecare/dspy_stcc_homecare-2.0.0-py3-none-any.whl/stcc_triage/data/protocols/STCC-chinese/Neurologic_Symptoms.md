# 神经系统症状

关键问题：姓名、年龄、起病时间、用药情况、既往史

其他考虑的协议：意识状态改变（28）；背部/颈部损伤（59）；困惑（150）；眩晕（199）；跌倒（240）；头痛（308）；头部受伤（311）；麻木和刺痛感（431）；疑似中风（576）；视力问题（639）；无力（649）。

护士提醒：

●突然的视力变化、无力、麻木、言语障碍或意识状态改变可能是中风或其他严重神经系统疾病的迹象。及时治疗可以防止大脑或脊髓受到广泛损害，并减少永久性残疾。用于溶解脑内血栓的药物必须在症状出现后3小时内使用。

●询问当前状况与正常情况有何不同。

提醒：记录呼叫者对建议、家庭护理指导以及何时回电的反应。

A. 下列任何症状是否突然发生？

●面部、手臂或腿部一侧麻木或无力

●不明原因的眩晕或跌倒

●呼吸困难

●意识状态改变

●无法站立、行走或承重

●言语障碍

●一侧面部下垂

●吞咽困难

●无法移动肢体

●视力变化

●突然剧烈头痛

●近期头部受伤史和血压升高

是 “拨打急救电话” 或 “立即寻求紧急医疗”

否 ↓ 转至 B

B. 下列任何症状是否存在？

●完全在数小时内缓解的短暂局灶性神经功能缺损

●新发且突然出现的膀胱或肠道失禁

●新发背痛和会阴区或肛门区麻木

●无法排尿且膀胱充盈

●头痛比之前的头痛更严重

●低血糖伴随无力、困惑、眩晕、头痛、颤抖或视力问题，并对常规家庭疗法无反应

是 “立即寻求医疗”

否 ↓ 转至 C

C. 下列症状是否存在？

●颤抖和有大量饮酒史

是 “24小时内寻求医疗”

否 ↓ 转至 D

D. 下列任何症状是否存在？

●注意力不集中

●手或脚的短暂刺痛感

●新发的颤抖，且无糖尿病、酒精滥用、帕金森病或癫痫发作史

是 “如果没有改善，请回电或预约主治医生” 并遵循家庭护理指导

否 遵循家庭护理指导



## 家庭护理指导

神经系统症状

●提供保证，即长时间坐姿或对区域施加压力后出现的脚或手部刺痛感在活动和拉伸后会消失。

●避免膝盖或踝关节交叉双腿。

●避免吸烟。

●按处方服药。

●在经历短暂症状时避免驾驶或操作机械。

●采取防跌措施。移除安全隐患。将铃铛或其他通讯设备放在伸手可及的地方，以便在尝试起床前呼救。

●不明原因的眩晕或跌倒

●呼吸困难

●意识状态改变

●无法站立、行走或承重

●言语或吞咽困难
