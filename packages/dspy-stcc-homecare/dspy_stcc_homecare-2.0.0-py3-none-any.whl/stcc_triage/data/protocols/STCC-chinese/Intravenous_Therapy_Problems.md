# 静脉治疗问题

关键问题 姓名，年龄，起始时间，原因，静脉（IV）的位置和类型，置入时间长度，用药情况，历史记录

其他需考虑的方案 伤口愈合与感染（664）

护士提醒：对于现有静脉通路的问题或疑问，请使用此方案。

提醒：记录来电者对建议、家庭护理指导以及何时回电的反应。

● 中心静脉导管明显脱出

● 中心静脉导管和突然出现呼吸困难或胸痛

B. 下列情况是否存在？

● 在家庭护理建议后仍无法冲洗中心静脉导管

● 家庭护理措施无效的持续发热

● 静脉插入部位发红、肿胀、疼痛、红线和温暖

● 中心静脉插入部位肿胀

是 “2到4小时内寻求医疗帮助” 并参阅家庭护理指导

否 ↓ 转到 C

C. 下列情况是否存在？

● 静脉无法冲洗，且没有管路或夹子问题

● 输液部位冷或肿痛

● 输液流速慢或不流动

● 捏住管路时无血液回流

● 管路中有血液积聚

是 “24小时内寻求医疗帮助” 并遵循家庭护理指导

否 ↓ 转到 D

D. 下列情况是否存在？

● 输液部位疼痛但没有肿胀或感染、渗漏的迹象

● 输液泄漏

是 “如果无改善，回电或联系PCP预约” 并遵循家庭护理指导

否 遵循家庭护理指导


## 家庭护理指导

静脉治疗问题

● 不要在输液部位上方戴手表或其他紧身首饰。

● 不要穿在输液部位上方过紧的衣物。

● 中心静脉导管问题：

● 检查管路是否有扭结和夹子。

● 改变身体或手臂的位置。

● 如果不在泵上，提高药袋位置。

● 检查泵是否正常工作。

● 如已接受冲洗指导，缓慢冲洗线路。如果线路容易冲洗，用10毫升生理盐水更有力地冲洗，然后重新开始输注。

● 如果线路在外部断裂，夹紧管路，并用橡皮筋固定以防止回流和凝结，联系PCP进行修复。

● 静脉问题：

● 如果输液流速慢或不流动且部位感觉冷，夹紧管路。

● 如果部位没有肿胀或疼痛，检查管路、泵和夹子是否有问题。改变肢体或身体的位置。如果不在泵上，提高药袋位置。

● 检查泵是否正常工作。泵是否已连接到电源（插电）？

● 如已接受冲洗指导，用3毫升生理盐水冲洗线路。

● 如果没有感染或渗漏的迹象，减慢输液速度，并在插入部位敷冷敷包。每10分钟检查一次部位是否有肿胀、发红或疼痛。

● 如果输液泄漏，干燥该区域并重新检查是否存在泄漏。紧固所有连接，更换帽子，如已接受更换管路指导，请更换和预充管路，然后重新开始输注。

● 如果输液脱出或拔出，在出血部位施加压力，并联系PCP重启输液。

● 如果家庭护理措施后仍持续泄漏，联系PCP重启输液。

附加指示

## 如有以下问题请联系您的PCP/诊所/急诊科

● 采取了家庭护理措施后问题仍然存在

● 渗漏迹象（部位冷、肿胀、发红、疼痛）

● 感染迹象（部位温暖、肿胀、发红、疼痛、红线）

● 持续不适

## 如果出现以下任何情况，请立即寻求紧急医疗帮助

● 输液部位疼痛，且正在输入抗癌药物

● 导管尖端可能断裂或折断

● 中心静脉导管明显脱出

● 中心静脉导管和突然出现胸痛或呼吸困难

如果来电者同意所给建议，记录通话，并鼓励来电者在问题恶化时回电或看PCP。

如果来电者不同意所给建议，重新评估并建议来电者与PCP、诊所或急诊科跟进。
