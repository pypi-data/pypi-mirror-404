# 西尼罗河病毒

关键问题 姓名、年龄、发病时间、病因、用药情况、病史（请考虑这些因素。）

其他需考虑的协议 精神状态改变 (28)；疲劳 (244)；成人发热 (250)，儿童发热 (253)；头痛 (308)；颈痛 (415)；神经系统症状 (418)；麻木和刺痛 (431)；成人皮疹 (500)，儿童皮疹 (505)；无力 (649)。

护士提示：只有在确诊为西尼罗河病毒或来电者对西尼罗河病毒有疑虑且症状出现在已知接触死鸟或蚊子叮咬后2至15天时，才使用此协议。

提醒：记录来电者对建议的反应、居家护理指导以及何时回电。

## 居家护理指导

西尼罗河病毒

●不要直接用手触摸死鸟。

●致电当地卫生部门 __________________ 获取如何报告和处理死亡鸟类进行测试和处置的指示。

●通过以下方式减少感染西尼罗河病毒的风险：

●使用驱虫剂；

●穿着长袖衬衫、长裤和袜子；

●尽量减少皮肤暴露；

●穿浅色衣物；

●避免在蚊子叮咬高峰期（从黄昏到黎明）外出；

●用纱窗覆盖敞开的门窗，并修理或更换有孔或不合适的纱窗；

●通过排空您财产上的物体或区域来避免积水；

●清理堵塞的雨水槽。

●休息。

●多喝水。

●服用您的常规止痛药以减轻不适。不要给儿童服用阿司匹林。如果年龄<20岁，避免使用类似阿司匹林的产品。如果有肝病，则避免使用对乙酰氨基酚。如果有肾病或胃部问题存在的情况下，或怀孕时，避免使用布洛芬。请按照标签上的说明服用。
