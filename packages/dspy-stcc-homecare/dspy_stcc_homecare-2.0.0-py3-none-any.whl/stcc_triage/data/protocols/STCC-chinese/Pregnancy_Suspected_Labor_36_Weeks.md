# 妊娠，疑似早产 <36周

关键问题：姓名、年龄、起始时间、原因、预产期、妊娠次数、孕龄、药物使用情况、病史

其他考虑的协议：妊娠，胎动减少（468）；妊娠，阴道流液（474）；妊娠，疑似早产（485）；妊娠，排尿问题（491）；妊娠，阴道出血（493）。

提醒：记录来电者对建议、家庭护理指导和何时回拨的反应。

## 评估与行动

A. 是否存在以下任一情况且孕周在20至36周之间？

●剧烈腹痛

●流液及脐带脱垂

●大量鲜红色阴道出血

●胎儿头部已显露即将分娩

是 “呼叫救护车” 或 “立即寻求紧急医疗”

否 ↓ 转到B

B. 是否存在以下任一情况？

●无胎动

●有宫颈环扎术史

●1小时内宫缩6次或更多

●阴道流液

●多胎妊娠且1小时宫缩4次或更多

●已知前置胎盘或部分剥离情况下任何宫缩

是 “立即寻求医疗”

否 ↓ 转到C

C. 是否存在以下任一情况？

●1小时内宫缩4至5次

●发热/寒战

●尿频、急迫性或疼痛性排尿

●首次出现宫缩时有当前的疱疹发作

●已知胎儿为臀位或横位情况下任何宫缩

●1小时胎动少于10次

●类似月经痛的痉挛

●骨盆压力感

●新发下背痛

●任何规律性疼痛

●阴道分泌物增多

●腹股沟疼痛

●有早产药物使用史

是 “2小时内寻求医疗”

否 ↓ 转到D

D. 是否存在以下任一情况？

●丢失黏液栓

●1小时少于4次无痛宫缩

是 “如无改善，回拨电话或联系初级保健医生（PCP）或产科护理提供者预约” 并遵循家庭护理指导

否 遵循家庭护理指导



## 家庭护理指导

妊娠，疑似早产 <36周

●左侧卧休息，并监测宫缩。

●补充水分。

●监测胎动。

●排空膀胱。

●避免乳头刺激和性生活。

额外指示












## 如遇以下问题，请联系您的初级保健医生/诊所/急诊科

●无胎动或1小时少于10次胎动
