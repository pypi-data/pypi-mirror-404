# 产后问题

关键问题 姓名、年龄、发病时间、分娩日期、用药情况、既往史

其他需考虑的方案 腹痛（成人）(9)；母乳喂养问题(98)；便秘(160)；抑郁(184)；阴道分泌物/疼痛/瘙痒(636)；伤口愈合和感染(664)

护士提示：产后可能出现多种情况。当存在多个相关症状时，应重点关注可能需要转诊至更高级别护理的首要问题。

提醒：记录患者对建议的反应、家庭护理指导以及何时回电。

## 评估与行动

A. 是否出现以下任何一种情况？

●呼吸困难或呼吸急促

●阴道出血需要每小时使用超过一片全尺寸卫生巾并伴有虚弱、头晕

●意识改变

●皮肤苍白、湿润且发凉

●分娩后48小时内下腹部剧烈疼痛

●有伤害婴儿或自己的想法

●剖宫产切口裂开

是 “叫救护车” 或 “立即寻求紧急医疗”

否 ↓ 转至B

B. 是否出现以下任何一种情况？

●体温>100°F (37.8°C) 在分娩后4到10天内

●伴有发热或腹痛的出血

●分娩第一周内出血量增加和腹部痉挛（<1小时内一片卫生巾被鲜红血液浸透）

●异常阴道分泌物伴有出血、发热、疼痛，或分泌物有恶臭

●呕吐、腹泻、发热或皮疹

●小腿肿胀、疼痛或发红

●比柠檬大的血块

●家庭护理措施无法缓解的头痛

●肌肉酸痛、发热和乳房上有红热疼痛区域

●寒战、发热和头痛

●阴道出血需要每小时使用超过一片全尺寸卫生巾

是 “2到4小时内寻求医疗”

否 ↓ 转至C

C. 是否出现以下任何一种情况？

●持续抑郁>2到3周

●排尿时的急迫感、频尿或疼痛

●无发热的异常阴道分泌物

●会阴切开术伤口处疼痛增加

●剖宫产伤口部位疼痛加剧、肿胀、恶臭引流、红热或温暖

是 “24小时内寻求医疗”

否 ↓ 转至D

D. 是否出现以下任何一种情况？

●乳房充血、触痛、硬结

●子宫收缩痛

●便秘

●会阴切开术部位的拉伸或牵拉感

是 “如果没有改善，请回电或预约初级保健医生” 并遵循家庭护理指导

否 遵循家庭护理指导



## 家庭护理指导

产后问题

●产后头两周平衡活动和休息。

●通常在6周检查后（剖宫产为8周）如果得到医生许可可以返回工作。

●记住，产后3到4天的子宫收缩是正常的，并且是由哺乳刺激产生的。

●排便可能在产后1到2天内不发生。每天喝6到8杯水和果汁并增加水果、蔬菜和麸皮的摄入量。轻泻剂（Metamucil）可能有帮助。请遵循说明书使用。

●直到引流停止前，每日更换敷料。

●移除无菌条带需在7到10天后进行。

产后抑郁（情绪低落）

●尽可能多休息，在婴儿睡觉时也休息。

●如散步等适度运动。

●保持均衡饮食。

●鼓励家人和朋友帮助准备餐食、家务劳动、婴儿照顾及清洁工作。

●谈论你的感受。

额外指导



## 如出现以下问题，请向初级保健医生/诊所/急诊科报告

●家庭护理措施无效或状况恶化

●发热、腹痛或异常阴道分泌物

●感染迹象：红肿加剧、疼痛、从伤口延伸出的红线、温暖感、恶臭或浓绿引流，或发热

●持续抑郁



## 如果出现以下任何情况，请立即寻求紧急医疗

●呼吸困难或急促

●每小时需要使用超过一片全尺寸卫生巾的阴道出血并伴有虚弱或头晕

●意识改变

●皮肤苍白且湿润

如果患者同意所给建议，请记录此次通话，并鼓励患者在问题恶化时回电或见初级保健医生。

如果患者不同意所给建议，重新评估并建议患者与初级保健医生、诊所或急诊科跟进。
