# 尿痛

关键问题 姓名，年龄，发病时间，过敏史（如怀孕超过20周，请参阅孕期尿路问题(491)），用药情况，病史，疼痛程度

需考虑的其他方案 成人腹痛(9)，儿童腹痛(13)；孕期尿路问题(491)；排尿困难(624)；尿液颜色异常(631)

提醒：记录来电者对建议的回应、家庭护理指导以及何时回电。

## 评估与行动

A. 是否存在以下任何情况？

●无法排尿

●尿中大量血迹

●腰背或腹部疼痛且超过8小时无法排尿

是 “立即寻求医疗帮助”

否 ↓ 前往B

B. 是否存在以下任何情况？

●最近进行过泌尿系统手术，出现意外的疼痛或发热

●生殖器或腹部近期受伤

●使用抗凝药物且尿液呈粉红色或红色

●排尿时极度不适

●腰背痛

●体温>102°F (39°C)

●阴囊严重疼痛或肿胀

●怀孕

●有红斑狼疮、肾小球肾炎或单肾病史

●会阴部或直肠疼痛

●腰背痛且体温>100.5°F (38.1°C)

●超过4小时无法排尿

是 “2到4小时内寻求医疗帮助”

否 ↓ 前往C

C. 是否存在以下任何情况？

●阴茎分泌物

●云状或恶臭的尿液（<2岁）

●儿童呕吐或液体摄入不足

●急迫感或频率超过3天

●膀胱部位疼痛

●云状或恶臭的尿液

是 “24小时内寻求医疗帮助”

否 ↓ 前往D

D. 是否存在以下任何情况？

●排尿结束时疼痛加剧

●反复出现泌尿系统感染

●性接触后尿痛

●频繁泡浴或生殖器区域有肥皂残留（尤其是在年幼的女孩中）

●急迫感或频率小于3天

是 “如果症状没有改善，请回电或预约家庭医生”

并

遵循居家护理指导

否 遵循居家护理指导


## 居家护理指导

尿痛

●多喝水，尤其是蔓越莓汁。

●避免摄入咖啡因和酒精。

●性交前后排尿。

●从前向后擦拭会阴部区域。

●穿棉质内衣。

●避免长时间穿着紧身裤。

●年幼女孩应避免频繁泡浴，并仅用水清洗会阴部。

●对于刺激或疼痛的会阴，可在洗澡水中加入半杯白醋并浸泡20分钟。2小时后再重复一次，12小时后再次进行。

●按常规服用止痛药（阿司匹林、对乙酰氨基酚、布洛芬）以缓解不适。不要给儿童使用阿司匹林。<20岁者避免使用类似阿司匹林的产品。肝病患者避免使用对乙酰氨基酚。肾病或胃部问题存在时，或孕期情况下避免使用布洛芬。请按照标签上的说明服用。

●如果反复出现泌尿系统感染和不适，请与药剂师讨论可用的非处方产品（如苯扎贝特盐酸盐[Pyridium]）。

## 如果以下问题出现，请向您的家庭医生/诊所/急诊科报告

●居家护理或抗生素治疗48小时后持续不适或状况恶化

●发热、腰背痛

●尿液中的血液增多

●阴囊严重疼痛或肿胀

●排尿困难

如果来电者同意所给建议，记录电话并鼓励其在问题加重时回电或就诊于家庭医生。

如果来电者不同意所给建议，请重新评估，并建议其与家庭医生、诊所或急诊科联系。
