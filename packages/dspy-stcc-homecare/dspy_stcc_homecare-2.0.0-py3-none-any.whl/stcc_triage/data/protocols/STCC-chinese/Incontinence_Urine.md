## 家庭护理指导

尿失禁

●练习加强和支持膀胱及生殖器官的肌肉。

●尝试在排尿中途收紧肌肉以停止尿流。持续计数到六，然后释放尿液再计数到六。停止并保持尿流计数到六，释放剩余尿液。

●每天多次坐着、站着或走路时练习控制尿液和肛门周围肌肉的收缩和放松。

●多喝水，包括蔓越莓汁，以帮助预防泌尿系统感染。

●使用护垫或吸水内裤。

●观察失禁的发生频率。避免可能引起或加重失禁的药物、食物或饮料（如咖啡因或酒精）。
