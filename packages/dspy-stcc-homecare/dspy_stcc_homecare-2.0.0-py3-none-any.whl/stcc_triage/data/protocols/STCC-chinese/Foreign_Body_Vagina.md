## 家庭护理指导

阴道异物

● 不要尝试移除阴道内的锐利物体或已经断裂的物体。

● 按照常规服用止痛药（阿司匹林、对乙酰氨基酚、布洛芬）。不要给儿童服用阿司匹林。10岁以下避免使用类似阿司匹林的产品。肝病患者避免使用对乙酰氨基酚。肾病或胃部问题存在，或怀孕时避免使用布洛芬。请按照标签指示服用。

● 注意感染的迹象：疼痛加剧、分泌物增多、发热或肿胀。

## 如出现以下问题，请联系您的家庭医生/诊所/急诊

● 多次尝试后仍无法取出异物

● 水肿和触痛持续或加重

● 异味排泄物或发热

● 皮疹、发热、手或脚脱皮、全身不适感、呕吐或腹泻
