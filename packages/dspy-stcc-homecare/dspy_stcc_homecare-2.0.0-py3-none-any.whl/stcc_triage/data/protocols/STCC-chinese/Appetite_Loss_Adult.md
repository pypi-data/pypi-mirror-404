关键问题：姓名、年龄、起始时间、用药情况、既往病史

其他需考虑的协议：成人腹痛（9）、酒精相关问题（21）、焦虑（36）、抑郁（184）、眩晕（199）、成人发热（250）、心率问题（320）、术后问题（457）、物质滥用、使用或暴露（5）、成人呕吐（642）。

●突然体重减轻>5至10磅

●数周内不明原因的体重减轻

●严重抑郁

●突然停止药物（包括处方药或非处方药）、酒精或咖啡因

否 ↓ 转D

●不良饮食习惯

●近期手术

●抑郁

●食欲减退新出现

●无其他症状

●活动减少

并

否 遵循家庭护理指导



## 家庭护理指导

成人食欲减退

●吃均衡的餐食。

●手术或疾病后缓慢增加食物量。

●定期锻炼。

●可尝试液体饮食补充剂（如Ensure、Boost）。向您的PCP或药剂师咨询其他非处方液态饮食补充剂。

●试图找出食欲减退的原因，并采取适当措施解决该问题（如假牙不合适、口腔溃疡、恶心、抑郁、吞咽困难、近期手术、化疗等）。

●提供安慰，告知在麻醉、抗生素和多种药物使用后的手术中出现食欲减退并不罕见。与您的外科医生或PCP讨论改善食欲及减缓体重下降的方法。

●服用常规的不适和发热药物。如存在肝病，则避免使用对乙酰氨基酚。如存在肾病、胃部问题或怀孕情况下，应避免使用布洛芬。按照标签上的指示使用。
