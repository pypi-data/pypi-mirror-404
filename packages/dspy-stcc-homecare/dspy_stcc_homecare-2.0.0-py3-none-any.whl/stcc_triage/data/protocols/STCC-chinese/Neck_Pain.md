# 颈痛

关键问题 姓名、年龄、发病时间、原因、相关症状、药物使用情况、病史、疼痛程度评分

其他需要考虑的协议 背部/颈部损伤（59）；胸痛（123）；腺体肿胀或触痛（297）；颌痛（387）；流行性腮腺炎（410）；麻木和刺痛感（431）

护士提醒：许多条件都可能引起颈痛。当颈痛伴随其他多个症状时，需谨慎评估并注意可能导致更严重状况的迹象，如脑膜炎（低头时疼痛加剧、头痛、发热、呕吐、困惑、畏光）或心脏病发作（胸痛、颈部、背部或颌部疼痛、出汗、心悸、恶心和/或呕吐）。

提醒：记录对建议的响应情况、家庭护理指导以及何时应回拨电话。

## 评估与行动

A. 颈痛是否与伤害有关？

是 转至背部/颈部损伤协议（59）

否 ↓ 继续B

B. 是否有胸痛？

是 转至胸痛协议（123）

否 ↓ 继续C

C. 疼痛突然发作，且存在以下任何症状吗？

●心脏病或心绞痛病史

●颌部疼痛

●出汗、心悸、恶心和/或呕吐

●呼吸困难

●低头时疼痛加剧并伴有以下任何情况：

●意识模糊/嗜睡

●严重头痛

●畏光

●发热

●紫色或血色皮疹

●双臂和双腿的麻木、刺痛或无力感

●排便或排尿控制变化

●头部不自主地转向一侧

是 “立即寻求紧急医疗”

否 ↓ 继续D

D. 存在以下任何症状吗？

●单侧手臂无力或麻木

●感染迹象：疼痛、肿胀、红斑、渗出物、温热感或红色条纹

●体温>103°F (39.4°C)

●体温>101°F (38.3°C) 且有糖尿病史、老年人、免疫抑制或静脉药物滥用者

●淋巴结肿大疼痛 >1英寸（2.5厘米）

●淋巴结肿胀疼痛并伴有吞咽困难

是 “在2到4小时内寻求医疗”

否 ↓ 继续E

E. 存在以下任何症状吗？

●皮疹

●颈痛伴随咽喉痛

●疼痛影响睡眠或活动

●颈部一侧或双侧肿胀

是 “在24小时内寻求医疗”

否 ↓ 继续F

F. 存在以下任何情况吗？

●睡觉姿势不正确

●新的运动或活动

●最近携带了重型包、手提包或其他物品，使用肩带

●侧面移动时疼痛加剧

●有长时间坐在电脑终端或工作站前的病史

●间歇性疼痛且有先前的颈部问题病史

是 “如果无改善，请回拨电话或预约家庭医生” 并遵循家庭护理指导

否 遵循家庭护理指导


## 家庭护理指导

颈痛

●每2小时使用热敷袋在颈部加热20分钟。糖尿病患者小心使用热疗，不要睡在电热垫上。

●睡觉时用毛巾折叠围绕颈部以减少颈部活动。

●携带手提包或公文包时放在手臂下，而不是肩上。

●避免长时间坐着。经常伸展和移动身体。

●定期锻炼以增强颈肌。

●泡热水澡或漩涡浴缸。

●按照医生的指示服用常用的止痛药（阿司匹林、对乙酰氨基酚、布洛芬）。不要给儿童服用阿司匹林。年龄<20岁者避免使用类似阿司匹林的产品。肝病患者避免使用对乙酰氨基酚。肾病或胃问题存在时，或者怀孕期间，避免使用布洛芬。遵循标签上的指示。

●保持良好的姿势和正确的身体对齐。

额外指导







## 向您的家庭医生/诊所/急诊室报告以下任何问题

●皮疹、发热或低头时疼痛加剧

●没有改善或病情恶化



## 如果出现以下情况，请立即寻求紧急医疗

●胸痛

●颈部、背部或颌部疼痛

●出汗、心悸、恶心和/或呕吐

●呼吸困难

●低头时疼痛加剧，意识模糊/嗜睡、严重头痛、畏光、发热、紫色或血色皮疹

●排便或排尿控制变化

如果来电者同意所给建议，记录通话并鼓励他们在问题恶化时回拨电话或看家庭医生。

如果来电者不同意所给建议，重新评估，并建议他们与家庭医生、诊所或急诊室跟进。
