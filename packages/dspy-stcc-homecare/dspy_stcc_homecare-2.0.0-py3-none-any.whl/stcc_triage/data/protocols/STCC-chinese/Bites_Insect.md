# 虫咬

关键问题 姓名，年龄，起始时间，虫类类型，过敏史，被咬部位，用药情况，疼痛评分，历史，是否开具了紧急过敏反应急救包

其他需考虑的协议 过敏反应（25）；床虫接触或疑虑（67）；蜜蜂蜇伤（72）；蜱虫叮咬（87）；荨麻疹（337）；瘙痒（382）；成人皮疹（500），儿童皮疹（505）；西尼罗河病毒（653）；伤口愈合与感染（664）

护士提示：过敏性休克是一种严重的、可能致命的过敏反应，可以在被蜇咬后几秒到一小时内出现。症状包括呼吸困难、感觉晕厥、舌头/喉咙/嘴唇肿胀、荨麻疹、喘息或咳嗽、或者有即将死亡的感觉。暴露于抗原后症状发生得越早，过敏性休克就越严重。

●如果开具了紧急过敏反应急救包，请遵照医生指导使用。注射通常在外侧大腿上最有效。请参阅家庭紧急护理指南以获取更多信息。

●如果有毒刺存在，应尽快移除，以减少毒素暴露。

提醒：记录对建议的响应、家庭护理说明以及何时回电。

## 评估与行动

A. 是否有以下症状？

●胸部紧绷或呼吸困难或吞咽困难

是 “呼叫救护车”

否 ↓ 转至B

B. 是否有以下情况？

●对同种昆虫有过严重过敏反应的历史

●被棕色隐士蜘蛛或黑寡妇蜘蛛咬伤

●精神状态改变

●在被咬后突然出现出汗和皮肤苍白

●舌头、喉咙或嘴唇肿胀

●嘴内蜜蜂蜇伤

●蝎子咬伤

是 “立即寻求紧急护理”

或者

“如果患者已经昏迷，失去意识，或者处于严重的呼吸窘迫中，请呼叫救护车”

并

遵循家庭紧急护理指南

否 ↓ 转至C

C. 是否有以下症状？

●被蜇部位以外的荨麻疹、皮疹、瘙痒或肿胀突然出现

●肌肉僵硬、腹痛和坐立不安

●恶心、呕吐或腹部痉挛

●多处被蜇

●有引流液、发热、红线或者除了红肿外还有脓包

●剧烈疼痛

是 “2到4小时内寻求医疗护理”

否 ↓ 转至D

D. 是否有以下症状？

●头痛、寒战、发热

●在被咬后大约48小时出现的感染迹象：加剧的疼痛、红肿、温热感、红线、引流或发热

否 ↓ 转至E

E. 请遵循家庭护理指南中的建议。

## 向您的初级保健医生/诊所/急诊报告以下问题

●没有改善或状况恶化

●有感染或过敏反应的迹象

●蜱虫叮咬后的皮疹、发热、头痛或关节痛

●荨麻疹、肌肉痉挛、肌肉无力或腹痛

## 如果出现以下情况，请立即寻求紧急护理

●精神状态改变

●呼吸困难或吞咽困难（呼叫救护车）

●胸部紧绷

●嘴唇/舌头/喉咙肿胀

如果求助者同意给出的建议，记录此电话，并鼓励求助者在问题恶化时回电或咨询初级保健医生。
