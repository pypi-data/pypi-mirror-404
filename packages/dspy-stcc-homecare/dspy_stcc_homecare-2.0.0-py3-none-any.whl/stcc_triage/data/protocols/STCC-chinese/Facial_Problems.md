## 评估与行动

A. 下列情况是否出现？

●突发视力丧失

●面部一侧严重疼痛，眼周模糊视觉和红眼

●成人肩部、胸部、颈部或臂部疼痛

●青光眼病史

●突然出现面部一侧下垂

是 “立即寻求紧急护理”

否 ↓ 转至 B

B. 下列情况是否出现？

●突发剧烈疼痛影响活动能力

●面瘫

●疼痛、肿胀、发红、发热或有分泌物

是 “2到4小时内就医”

否 ↓ 转至 C

C. 下列情况是否出现？

●午后或弯腰时疼痛加剧

●鼻腔流出绿色、棕色或黄色的分泌物

●鼻子与下眼睑之间的棱线上疼痛

●体温 >101°F (38.3°C)

●持续肿胀

●面部皮疹、水泡或病变

是 “24小时内就医”

否 ↓ 转至 D

D. 下列情况是否出现？

●近期面部出现红色水疱状皮疹

●面部受伤后的疼痛、肿胀或瘀伤

●近期感冒史

●食用冰冻食物或饮料后疼痛加剧

●鼻腔和眼部有分泌物

是 “如无改善，请回电或预约家庭医生”

并遵循家庭护理指导

否 遵循家庭护理指导



## 家庭护理指导

面部问题

●交替使用冷热敷于前额和脸颊，每次1分钟，每天4次。袜子内装满在微波炉中加热的米粒效果很好。

●增加水分摄入量。

●对于面部伤害，在最初24小时内每天4次将冰袋敷于脸上10到20分钟以帮助减少肿胀。不要直接将冰放在皮肤上，应在冰和皮肤之间放置湿布或其他织物屏障。

●每天多次坐在蒸汽浴室中20分钟以促进鼻窦排液。

●根据需要服用非处方解充血剂以缓解充血，并遵循标签上的指示。向药剂师咨询产品建议。

●按照常规剂量（阿司匹林、对乙酰氨基酚、布洛芬）服用止痛药以缓解不适和发热。不要给儿童使用阿司匹林。年龄<20岁者避免使用类似阿司匹林的产品。肝病患者避免使用对乙酰氨基酚。肾病或胃部问题存在时，或怀孕情况下避免使用布洛芬。遵循标签上的指示。

额外指导



## 如果出现以下问题，请向您的家庭医生/诊所/急诊报告

●持续性疼痛或病情恶化

●体温 >101°F (38.3°C)

●感染迹象：疼痛、肿胀、发红、发热、分泌物或红线

●持续性鼻腔分泌物

●视力、听力、嗅觉或味觉变化



## 如果出现以下情况之一，请立即寻求紧急护理

●突发视力丧失

●面部一侧严重疼痛，眼周模糊视觉和红眼

●成人肩部、胸部、颈部或臂部疼痛

●面部一侧突然下垂

如果对方同意所给的建议，记录通话，并鼓励对方如病情恶化时回电或就医。

如果对方不同意所给的建议，重新评估并建议对方跟进家庭医生、诊所或急诊。
