# 皮疹，儿童

关键问题：姓名、年龄、发病时间、原因、部位、用药情况、既往病史、免疫接种状况、伴随症状

其他需考虑的方案：过敏反应（25）；床虫暴露或疑虑（67）；蜂蜇伤（72）；昆虫叮咬（79）；水痘（129）；尿布疹（190）；热暴露问题（323）；荨麻疹（337）；瘙痒（382）；风疹（520）；麻疹（523）；疥疮（527）；皮肤病变：肿块、包和溃疡（556）；纹身问题（604）；附录M（696）。

护士提醒：

●有许多条件会导致皮疹。当皮疹伴随其他多个症状时，应使用主要关注点且转介至更高护理级别的概率最高的方案。

●附录M（696）中的传染病表旨在帮助护士更好地了解可引起皮疹的几种传染病。不应尝试用此表格诊断呼叫者的情况。

提醒：记录呼叫者对建议、家庭护理措施的反应，以及问题恶化时是否回电或就诊医生。

## 家庭护理指导

儿童皮疹：

●尽量找出原因并避免接触刺激物。

●用肥皂和水清洁受影响区域以去除刺激物，然后仅用水清洗该区域。

### 控制瘙痒

●每天可多次在浴缸中加入一杯小苏打、艾维诺或燕麦片，洗凉水澡，或者每3-4小时对局部皮疹使用20分钟的冷敷包。

●对于严重瘙痒，可以涂抹1%氢化可的松霜并遵循标签说明。冷藏以获得最佳冷却效果。在怀疑股癣、脚气、脓疱疮或环形虫感染时避免使用。

●将白醋和小苏打混合制成膏状物，或使用柜台出售的炉甘石洗剂或艾维诺涂抹受影响区域。

●给儿童服用抗组胺药（如苯海拉明、氯雷他定）并遵循标签说明。

●剪短孩子的指甲，并劝阻其抓挠。

●用袜子覆盖婴儿的手以减少抓挠。

●使用布洛芬溶液浸泡的湿敷料，1份溶液加入10至40份水中。频繁更换，每2小时内最多8次。

### 可能的过敏反应

●如果皮疹与新药有关，请停止服用该药物。如果是处方药，请咨询医生。

●按照容器上的指示服用抗组胺药直到皮疹和瘙痒消失。

●注意观察反应恶化迹象（如肿胀或吞咽、呼吸困难）并联系医生。

### 可能的热疹

●涂抹炉甘石洗剂或氢化可的松霜。

●每2-3小时根据需要不使用肥皂淋浴或洗澡以缓解症状，并自然晾干。

●在受影响区域涂抹婴儿爽身粉。

### 毒橡树、毒葛或毒漆藤接触

●尽可能在接触后1小时内清洗受影响区域。

●用凉水浸泡该区域或擦冰20分钟，根据需要进行。

●清洗衣物和任何暴露于植物的动物。

### 怀疑麻疹/水痘/风疹

●让儿童待在家里直到皮疹消失以避免将疾病传播给他人。

●避免将感染扩散给其他人。

●接触感染者后用肥皂和水彻底洗手。

●当有可能与人接触时，保持渗出区域覆盖。

●使用棉球或纱布涂抹药膏或软膏到开放部位。

附加指导

## 出现以下问题时向您的医生/诊所/急诊报告

●24-48小时内家庭护理措施无改善或状况恶化

●发烧、喉咙痛或关节疼痛

●感染迹象

●异常嗜睡、拒绝饮水、耳痛和呼吸急促

### 立即寻求紧急医疗帮助，如果出现以下情况

●呼吸困难、胸闷或喉咙后部肿胀

●紫色斑点、头痛、颈部僵硬、呕吐或发烧

●皮疹、发烧、红舌和淋巴结肿大

●肛门区域红色脱皮性皮疹

如果呼叫者同意所给建议，记录通话并鼓励其在问题恶化时回电或就诊医生。

如果呼叫者不同意所给建议，重新评估并建议其与医生、诊所或急诊联系。
