# 皮肤病变：肿块、突起和溃疡

关键问题 姓名，年龄，发病时间，过敏史，用药情况，病史

## 评估与行动

A. 是否存在以下任何一种情况？

●面部或眼睛附近，直肠或生殖器上新出现的大病变

●剧烈疼痛

●疼痛性水泡

●发热

●最近使用了磺胺类药物

●糖尿病患者或免疫系统受损且病灶有感染迹象

是 “2到4小时内寻求医疗护理”

否 ↓ 转至B

B. 是否存在以下任何一种情况？

●病变位于上唇附近、鼻尖或鼻孔内，眼睛或眼睑上

●体温 >100°F (37.8°C)

●最近痣的颜色、形状或大小迅速变化

●疼痛性或出血性的痣

●感染迹象：疼痛、肿胀、红斑、分泌物、红线或温暖感

●足底出现使行走困难的病变

●家庭护理治疗48至72小时后持续存在的触痛病变

●腋下有突起

●身体其他部位新发病变

●先感到疼痛，随后出现带状、条纹或小面积的皮疹和水泡

是 “24小时内寻求医疗护理”

否 ↓ 转至C

C. 是否存在以下任何一种情况？

●突出到经常被碰撞或妨碍的程度的疣或痣

●请求去除疣

●有反复出现脓肿、疣或病变的历史

●治疗两周后新发疣

●治疗8周后持续存在的疣

●突发丘疹

●半英寸至一英寸，中心清晰且瘙痒的环状、隆起、粗糙的粉红色斑块

●头皮上半英寸至一英寸，中心清晰且瘙痒的环状、隆起、粗糙的粉红色斑块

●可能为性传播疾病的担忧

●家庭护理治疗10天后持续存在的无痛性病变

●反复出现的病灶，个人请求开具预防未来发作的处方

是 “若没有改善，请回电或预约家庭医生（PCP）”

## 向您的家庭医生/诊所/急诊室报告以下问题

●家庭护理治疗72小时后持续存在的生长、变化、出血、颜色变化、疼痛或愈合不良

●感染迹象：疼痛、肿胀、红斑、分泌物、红线或温暖感

●请求去除皮赘、痣或疣

如果来电者同意所给的建议，记录通话并鼓励来电者在问题恶化时回电或就医。如果来电者不同意所给的建议，重新评估，并建议来电者与家庭医生（PCP）、诊所或急诊室联系跟进。
