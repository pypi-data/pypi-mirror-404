护士提示：仅在已确诊关节炎并正在接受治疗且有关于关节炎或关节炎症状未对常用药物或治疗产生反应的问题时使用此方案。最常见形式的关节炎包括：

●退行性关节炎或骨关节炎：主要影响老年人，多发于手指、颈部、髋部、膝盖和下背部。通常由磨损或创伤后损伤引起。

●幼年特发性关节炎：无外伤或感染情况下持续超过6周的儿童（<16岁）关节炎症。初期全身起病，伴随间歇性高烧>39°C (102.2°F) 持续>12周后出现皮疹和多达四个关节疼痛。早期迹象包括跛行和类似流感的症状。

●类风湿性关节炎（RA）：肿胀、疼痛的关节和疲劳症状逐渐出现，持续数周至数月。早晨症状更严重，首先影响手足小关节，然后进展到肘部，最终累及承重关节。

●化脓性关节炎：单个关节突然剧烈疼痛、肿胀、发热和活动时严重疼痛。通常影响膝盖、髋部、肩部或踝部。在静脉注射毒品使用者（IVDUs）、动物或人类咬伤、使用类固醇、细菌、病毒或真菌感染中也较为常见。化脓性关节炎需要紧急评估。

## 报告以下任何问题给您的主治医生/诊所/急诊

●没有改善或病情恶化
●疼痛加剧
●活动能力下降
●发热
●高烧>102.2°F (39.0°C) 持续超过2周，皮疹或关节疼痛对当前处方药物无反应
●关节功能丧失
●突然出现的严重膝部、小腿或大腿疼痛，肿胀、温热且无明显外伤
●单个关节突然剧烈疼痛、发热、肿胀和温热

## 如果发生以下任何情况立即寻求紧急医疗护理

●踝部肿胀伴有胸痛、咳血或呼吸困难
●受影响部位的手指或脚趾变冷或发蓝
