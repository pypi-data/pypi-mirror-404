# 性传播疾病 (STD)

关键问题：姓名、年龄、发病时间、疑似性病、已知或可疑的性病暴露史、用药史

考虑其他方案：血液/体液暴露（93）；生殖器病变（291）；男性生殖器问题（294）；性侵犯（539）；排尿疼痛（628）；阴道分泌物/疼痛/瘙痒（636）。

护士提示：如果已知或怀疑有性病暴露，使用此方案。附录N（687）中的《性传播疾病表》旨在帮助护士更好地了解多种性病的传播方式、潜伏期和传染期。该表格不可用于尝试诊断来电者的情况。

提醒：记录对建议的反应、家庭护理指示及何时回拨。

## 家庭护理指示

性传播疾病 (STD)

●温水浴可缓解疱疹的不适，但不能治愈。在有开放性溃疡时避免性行为。始终使用乳胶避孕套。

●如果与多个伴侣发生无保护措施的性行为或与已知或疑似HIV感染者发生性行为，请立即进行HIV感染实验室测试，并在初次暴露后的3个月和6个月再次检测。为防止自己和其他人被感染，应使用避孕套或避免性行为。联系家庭医生或卫生部门诊所以获取检测服务。接触后可能有即时的医疗治疗。

●双方都应接受性病检测并接受治疗，且应避免无保护措施的性接触。

●在进行亲密性行为前，与新伴侣讨论性病和HIV暴露史。

●使用乳胶避孕套（除非对乳胶过敏）。

额外指示



## 如果出现以下问题，请联系您的家庭医生/诊所/急诊室

●有感染HIV的风险且持续疾病、疲劳、体重下降、腹泻、淋巴结肿大、溃疡、干咳或夜间盗汗

●怀疑暴露于性病

●异常的阴道或阴茎分泌物、生殖器病变或生殖器疼痛

●已知生殖器疱疹且无法忍受持续不适

●盆腔疼痛（伴或不伴有发热）



## 如果出现以下任何情况，请立即寻求紧急护理

●高烧并已知带状疱疹发作

●新发混乱或谵妄

●眼睛周围或内部的疼痛、红肿或皮疹

如果来电者同意所给建议，记录通话，并鼓励其在问题恶化时回拨电话或就诊家庭医生。如果来电者不同意所给建议，请重新评估并建议其向家庭医生、诊所或急诊室跟进。
