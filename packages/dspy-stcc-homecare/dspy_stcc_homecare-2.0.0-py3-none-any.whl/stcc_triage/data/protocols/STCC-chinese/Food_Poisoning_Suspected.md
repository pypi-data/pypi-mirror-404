# 涉嫌食物中毒

关键问题 姓名、年龄、发病时间、伴随症状、病史、用药情况、疑似来源

其他需考虑的协议 腹痛，成人（9），儿童（13）；腹部膨胀（16）；禽流感“鸟流感”接触（55）；脱水（180）；腹泻，成人（192），儿童（195）；直肠出血（510）；呕吐，成人（642），儿童（645）。

护士提醒：如果来电者在餐厅就餐后怀疑食物中毒，请鼓励来电者通知餐厅。其他顾客可能也遇到了同样的问题。早期通知有助于餐厅追踪来源并解决问题。

提示：记录来电者对建议的反应、家庭护理指导以及何时回电。

## 评估与行动

A. 食用罐装、熏制或真空包装食品后6至48小时出现不适，且存在以下任何情况？

●精神状态改变

●呼吸困难、吞咽困难或视力模糊

●呕吐带血

●便中带血

是 “立即寻求紧急医疗帮助”

否 ↓ 继续B

B. 存在以下任何情况吗？

●疼痛严重、加剧或持续>4小时

●恶心和呕吐超过12小时且对家庭治疗或非处方药无反应

●腹泻超过48小时且对家庭治疗或非处方药无反应

●便中带血

●脱水迹象：

●尿量减少

●眼睛凹陷

●皮肤松弛干燥

●极度口渴、口腔干燥

●孩子哭泣时没有眼泪

●站立时头晕

●皮疹

●发热

是 “在2到4小时内寻求医疗帮助”

否 ↓ 继续C

C. 存在以下任何情况吗？

●食用未冷藏的肉类、禽类、鱼类、蛋黄酱或鸡蛋制品后出现恶心、呕吐、腹泻或腹痛

●同餐的其他人也感到不适

是 “如果没有改善，请回电或预约家庭医生”

否 遵循家庭护理指导



## 家庭护理指导

疑似食物中毒

●频繁（每5到10分钟）小口喝清液（苹果汁、茶、肉汤、运动饮料、透明汽水、姜汁汽水），直到恶心和呕吐缓解。在呕吐缓解后的头24小时内，根据耐受情况逐渐增加量。

●腹泻时避免食用乳制品。

●呕吐和腹泻缓解后48小时内避免食用辛辣食物、酒精、乳制品和咖啡。

●避免服用阿司匹林。

●预防措施：

●避免食用已放置超过2小时的肉类、调料、酱料和蛋黄酱基食品。保持食品温度<40°F（4.4°C）或>140°F（60°C）。

●不要食用盖子鼓起的罐头或瓶装内容物。

●在冰箱或微波炉中解冻肉类，而不是在室温下解冻。

●频繁洗手、切菜板和台面，尤其是在处理生鸡肉或鸡蛋后。

●如果食物有异味或发臭，请勿食用。

额外指示









## 出现以下问题时报告给您的家庭医生/诊所/急诊

●症状持续>48小时或恶化

●粪便或呕吐物中带血（不仅仅是条纹或斑点）

●体温>101°F（38.3°C）



## 如果出现以下任何情况立即寻求紧急医疗帮助

●视力模糊

●呼吸困难

●吞咽困难

●意识水平下降

●脱水迹象

如果来电者不同意所给的建议，重新评估并建议来电者跟进家庭医生、诊所或急诊。
