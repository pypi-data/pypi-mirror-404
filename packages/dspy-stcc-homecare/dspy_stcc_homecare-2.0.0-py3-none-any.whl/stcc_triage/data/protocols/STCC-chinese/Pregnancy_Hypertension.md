# 妊娠，高血压

关键问题：姓名、年龄、发病时间、病因、孕周、妊娠次数、病史、用药情况

其他需要考虑的方案：妊娠，胎动异常（468）；妊娠并发症（481）；妊娠，疑似临产（485）；妊娠，小于36周疑似临产（488）。

护士提示：如果孕妇正在接受已知妊娠期高血压治疗，请使用本协议。

提醒：记录来电者对建议的反应、家庭护理指导以及何时回电。

## 评估与行动

A. 是否存在以下任何情况？

●抽搐活动

●惊厥

●无反应

●孕20周以上且有：

●剧烈腹痛

●显著阴道出血

●即将分娩，胎儿头部露出

是 “叫救护车”或“立即寻求紧急医疗”

否 ↓ 转到B

B. 是否存在以下任何情况？

●鲜红色阴道出血

●剧烈头痛

●无胎动

●手、脸、腿或腰部突然肿胀加剧

●上腹痛（右上象限）

●视觉障碍（视物模糊或双重视觉，眼前有黑点，或看到星点或物体周围光环）

是 “立即寻求医疗”

否 ↓ 转到C

C. 是否存在以下任何情况？

●家庭护理措施无法缓解的头痛

●1小时内胎动少于10次

●规律性强收缩

●腹痛

●血压升高（在家监测后休息时）

●呕吐且不能摄入液体

是 “2至4小时内寻求医疗”

否 ↓ 转到D

D. 是否存在以下任何情况？

●脚或腿逐渐肿胀

●家庭护理措施可缓解的头痛

●每周体重增加>2磅

●恶心

是 “如果没有改善，请回电或联系初级保健医生或妇产科医生预约”并遵循家庭护理指导

否 遵循家庭护理指导


## 家庭护理指导

妊娠，高血压

●左侧卧休息。

●监测胎动；如果1小时内胎动少于10次，寻求医疗帮助。
