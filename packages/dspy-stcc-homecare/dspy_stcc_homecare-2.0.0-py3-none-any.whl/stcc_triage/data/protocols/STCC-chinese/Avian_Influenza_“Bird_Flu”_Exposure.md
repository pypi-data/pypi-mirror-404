# 禽流感（“禽流”）暴露

关键问题 姓名，年龄，发病时间，过去10天内是否前往疑似地点（已知的家禽或人类中存在禽流感感染），最近是否有接触过家禽或家养鸟类（如家禽农场、养有家禽的家庭或鸟市），频繁接触死亡家禽的情况，当前所在地近期是否发生疫情及可能暴露情况，所用药物，既往病史

其他需考虑的协议 呼吸困难（106）；普通感冒症状（146）；鼻塞（153）；咳嗽（170）；脱水（180）；眼部问题（228）；成人发热（250），儿童发热（253）；咽喉疼痛（564）。

护士提示：如果怀疑暴露于禽流感，请使用此协议。潜伏期为3至5天。禽流感通常感染鸟类，较少情况下会感染猪和人类。

提醒：记录来电者对建议的反应、家庭护理指示以及何时需要回电。

## 评估与行动

A. 下列任何情况是否存在？

●出汗且伴有头晕或虚弱感

●严重呼吸困难

●面色苍白或发灰，并有病毒性肺炎病史

●意识混乱、谵妄或难以唤醒

是 “呼叫救护车” 或 “立即寻求紧急医疗”

“通知护理人员潜在暴露”

否 ↓ 继续到B

B. 下列任何情况是否存在？

●可能接触过受污染的鸟类，包括被分泌物/排泄物污染的表面，并伴有结膜炎和发热、咳嗽、喉咙痛或身体疼痛等症状

●不褪色的暗红色或紫色皮疹

●照顾已知患有禽流感的人后出现结膜炎

●儿童体温 >105°F (40.6°C)

●成人体温 >103°F (39.4°C)

●有免疫抑制病史、年龄>60岁、镰状细胞贫血、糖尿病或卧床不起且体温 >101°F (38.3°C) 的情况

●婴儿 <3 个月大，体温 >100.4°F (38°C)

●老年人或免疫力低下者出现脱水迹象

是 “立即就医” 并 “通知护理人员潜在暴露”

否 ↓ 继续到C

C. 下列任何情况是否存在？

●照顾已知患有禽流感的人并伴有类似流感的症状或结膜炎

●喉咙痛持续超过2天或发热 > 2 天

●发热且有哮喘、癌症、慢性阻塞性肺病（COPD）、充血性心力衰竭（CHF）、糖尿病、心脏病或肾病史

●绿色、棕色或黄色痰液或鼻分泌物 >72 小时

●脱水迹象

是 “在24小时内就医”

否 ↓ 继续到D

D. 下列任何情况是否存在？

●绿色、棕色或黄色痰液或鼻分泌物 <72小时

●痰中带血丝

●对潜在的禽流感暴露有担忧但无其他症状（或轻度症状）

是 “如果没有好转，请回电或预约家庭医生”

并遵循家庭护理指示

否 遵循家庭护理指示



## 家庭护理指示

禽流感（“禽流”）暴露

●出现类似流感症状的人应在最后一次发热后24小时内留在家中（除非看医生）。

●接触患者及其环境（包括洗衣物）的护理人员应经常用肥皂和水或含酒精的手部消毒剂洗手。

●咳嗽和打喷嚏时要遮住口鼻，并将纸巾丢入袋子中。

●脏盘子和餐具应用洗碗机清洗或使用温水和肥皂手洗。

●避免接触死亡的鸟类或被分泌物/排泄物污染的表面。

额外指示









## 出现以下问题应向您的家庭医生/诊所/急诊报告

●护理人员出现类似流感的症状

●喉咙痛持续超过2天或发热 > 2 天

●绿色、棕色或黄色痰液或鼻分泌物 >72 小时

●脱水迹象

●没有好转或病情恶化



## 如果发生以下任何情况应立即寻求紧急医疗帮助

●出汗且伴有头晕或虚弱感

●严重呼吸困难

●面色苍白或发灰，并有病毒性肺炎病史

●意识混乱、谵妄或难以唤醒

●胸痛
