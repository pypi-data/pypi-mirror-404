# 毒物中毒，疑似

关键问题 姓名、年龄、起病时间、原因、既往史、相关症状、用药情况

其他考虑的方案 抑郁症（184）；成人腹泻（192），儿童腹泻（195）；食物中毒，疑似（262）；过量（438）；物质滥用、使用或暴露（582）；自杀企图，威胁（585）；成人呕吐（642），儿童呕吐（645）。

护士提醒：如果已知或怀疑中毒，请使用此方案。询问吞下了什么，多少，何时吞下，以及任何相关症状。询问其他可能也暴露于此但未主动告知的孩子们，因为他们担心可能会受罚而不愿意透露信息。

提示：记录来电者对建议的回应、居家护理指示及何时回电。

## 评估与行动

A. 是否存在以下任一情况？

●意识状态改变或无反应

●严重呼吸困难或呼吸频率<10次/分钟

●胸痛

●自杀企图

●惊厥发作

●意识水平变化

是 “呼叫救护车并带容器和物质前往医院。” 如果没有脉搏或呼吸，指导开始CPR或急救呼吸。

否 ↓ 继续B

B. 是否存在以下任一情况？

●大量出汗或唾液分泌过多

●喘息或呼吸困难

●嘴唇、口腔或指甲床发绀

是 “立即寻求紧急医疗并带容器和物质前往医院”

否 ↓ 继续C

C. 是否存在以下任一情况？

●恶心、呕吐、腹泻

●腹痛

●唇部、舌头或皮肤烧伤

●心悸

●头痛、烦躁或发热

●有精神疾病史

是 “立即寻求医疗” 并遵循居家护理指示

否 ↓ 继续D

D. 是否存在以下任一情况且无症状？

●儿童周围有敞开的溢出容器、药瓶，或面部、皮肤、衣物上有物质

●呼吸或衣物上可闻到产品气味

●怀疑摄入了阿司匹林、对乙酰氨基酚或其他非处方药物、致幻剂或未知蘑菇

是 “呼叫毒物控制中心”（电话号码：1-800-222-1222）并遵循居家护理指示

否 “如果情况没有改善，请回电或预约家庭医生” 并遵循居家护理指示



## 居家护理指示

疑似中毒

●按照毒物控制中心的指示行事。向中心提供以下信息：

●确认摄入的物质。从容器标签上读取准确名称。

●描述摄入量或容器中缺失的数量。

●指出物质被摄入的时间。

●阅读容器上的意外摄入说明。

●带容器、剩余内容物、植物或蘑菇前往医院。

●如果无法及时联系到毒物控制中心，请呼叫救护车。

●不要在人员意识状态改变或吞咽困难时诱导呕吐。除非毒物控制中心指示，否则不要诱导呕吐。以下物质摄入后不应诱导呕吐：酸性、碱性或石油产品；电池、硫酸或盐酸；漂白剂；下水道或烤箱清洁剂；汽油；家具抛光剂；煤油；打火机燃料。

●除非毒物控制中心认为无需担心，否则所有过量情况都应由医生评估。

●不要试图通过将受害者放在淋浴中或强迫其喝咖啡来唤醒。

●未经毒物控制中心指示，不要给予任何食物或饮料。

## 如遇以下问题，请报告给毒物控制中心

●初次治疗后的持续问题或疾病

●不确定是否摄入了物质且出现恶心或呕吐



## 如果发生以下任一情况，立即寻求紧急医疗

●大量吞咽或唾液分泌过多

●喘息或呼吸困难

●嘴唇、口腔或指甲床发绀

如果来电者同意所给建议，请记录通话并鼓励其在问题恶化时回电或就诊家庭医生。

如果来电者不同意所给建议，请重新评估，并建议其与家庭医生、诊所或急诊科跟进。
