# 腺体，肿胀或触痛

关键问题 姓名、年龄、发病时间、过敏史、用药情况、病史、位置

其他需考虑的协议 流行性腮腺炎（410）；风疹（520）；麻疹（523）；皮肤病变：肿块、凸起和疮口（556）。

提醒：记录来电者对建议的反应，家庭护理指导及何时回电。

## 评估与行动

A. 是否存在以下任何情况？

●肿大的淋巴结大于2英寸

●肿大淋巴结附近有红色条纹

●淋巴结处发热或发红

●剧烈疼痛

●淋巴结影响呼吸、吞咽或颈部活动

●淋巴结流脓

是 “请在2到4小时内寻求医疗照顾”

否 ↓ 转至B

B. 是否存在以下任何情况？

●肿大的淋巴结1到2英寸大小

●持续发热且退热措施无效

●腹股沟、后颈或腋窝区域有肿大的淋巴结

●肿大淋巴结附近或远端区域的感染迹象

●身体其他部位的持续性皮疹、瘙痒或肿胀

●持续疲劳

●夜间盗汗

是 “请在24小时内寻求医疗照顾”

否 ↓ 转至C

C. 是否存在以下任何情况？

●间歇性的淋巴结肿大史

●伴有发热、咽喉痛或鼻塞的颈部淋巴结肿大

●有过敏史

●月经前腋下区域肿胀和触痛

是 “如果无改善，请回电或预约家庭医生”

并

遵循家庭护理指导

否 遵循家庭护理指导



## 家庭护理指导

腺体，肿胀或触痛

●不要挤压肿大的淋巴结。

●根据需要服用常规止痛药（阿司匹林、对乙酰氨基酚、布洛芬）以缓解不适或发热。不要给儿童使用阿司匹林。20岁以下者避免使用类似阿司匹林的产品。肝病患者避免使用对乙酰氨基酚。肾病、胃部问题患者或孕妇避免使用布洛芬。遵循药物说明书上的指示。

●注意感染的迹象。

其他指导



## 如有以下问题，请告知您的家庭医生/诊所/急诊科

●淋巴结影响呼吸、吞咽或颈部活动

●月经后腋下持续肿胀或触痛

●持续的淋巴结肿大

●感染迹象：红斑增多、疼痛加剧、肿胀、发热、红色条纹、流脓

如果来电者同意所给建议，记录通话并鼓励其在问题恶化时回电或就诊家庭医生。

如果来电者不同意所给建议，重新评估，并建议与家庭医生、诊所或急诊科联系。
