# 妊娠问题

关键问题：姓名、年龄、发病时间、妊娠期、孕次、病史、用药情况

其他考虑的协议：腰背痛（62）；乳腺问题（103）；便秘（160）；成人腹泻（192）；足部问题（265）；头痛（308）；胃灼热（316）；痔疮（327）；妊娠期高血压（471）；妊娠阴道流液（474）；妊娠期恶心呕吐（478）；疑似早产（485）；<36周的疑似早产（488）；妊娠排尿问题（491）；妊娠阴道出血（493）；肿胀（597）。

护士提示：妊娠期间可能出现许多状况。当有多个相关症状时，应重点关注最有可能需要转诊至更高护理级别的主要问题。

提醒：记录来电者对建议的反应、家庭护理指导和何时回电。

## 评估与行动

A. 妊娠期>20周，并且以下任何情况是否存在？

●即将分娩，胎儿头部露出

●严重头痛、视力重影或模糊、方向感丧失、头晕或易怒

●二尖瓣区域先前受过创伤、跌倒、腹部受到撞击等

●排出大血块

●抽搐

●剧烈腹痛

●阴道流液伴随脐带脱垂

●诊断为子痫前期并出现新症状

是 “叫救护车” 或 “立即寻求紧急护理”

否 ↓ 转至B

B. 以下任何情况是否存在？

●无胎动

●面部、手部、腿部或腰部突然肿胀

●头痛、眼前黑点、头晕、昏厥或呕吐

●规律宫缩或阴道流液

是 “立即就医”

否 ↓ 转至C

C. 以下任何情况是否存在？

●已知高血压且血压升高

●腿痛、肿胀、发红或负重时疼痛加剧

●疼痛、阴道出血或发热

●对止痛药无反应的持续头痛

●所有食物和液体及/或腹痛持续呕吐

●1小时内少于10次胎动

是 “2小时内就医”

否 ↓ 转至D

D. 以下任何情况是否存在？

●急迫的排尿感、疼痛或困难

●尿液颜色异常（如深黄、红色或棕色）

●无法解释的体重减轻

●疲劳或虚弱加剧

●阴道出血量增多

是 “立即就医”

否 ↓ 转至E

E. 以下任何情况是否存在？

●持续性头痛、头晕或视力问题

●呼吸困难或气短

●胸痛或不适

●心悸或心跳加速

●极度焦虑或情绪波动

是 “立即寻求紧急护理”

否 “记录并继续观察，如症状加重请回电或就医”

## 报告以下任何问题给您的PCP/诊所/急诊科

●采取家庭护理措施后情况未改善或恶化

●1小时内少于10次胎动

## 如出现以下任何情况立即寻求紧急护理

●即将分娩

●严重头痛、视力重影或模糊、方向感丧失、头晕

●排出大血块

●剧烈腹痛

●阴道流液伴随脐带脱垂

●面部、手部、腿部或腰部突然肿胀

●头痛、眼前黑点

●规律宫缩或阴道流液

●无胎动

如果来电者同意所给的建议，记录电话并鼓励在问题恶化时回电或就医。

如果来电者不同意所给的建议，重新评估，并建议其联系PCP、诊所或急诊科。
