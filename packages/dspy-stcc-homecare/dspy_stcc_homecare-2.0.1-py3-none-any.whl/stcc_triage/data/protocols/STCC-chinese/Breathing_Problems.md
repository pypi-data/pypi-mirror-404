# 呼吸问题

关键问题：姓名，年龄，发病时间，原因，用药情况，既往史

其他需考虑的方案：过敏反应（25）；哮喘（52）；胸痛（123）；慢性阻塞性肺疾病（COPD）（138）；窒息（135）；普通感冒症状（146）；充血（153）；心力衰竭（CHF）（157）；咳嗽（170）；吸入异物（271）；过度换气（350）；喘息（657）。

护士提示：如果已知有呼吸问题，使用处方吸入器、氧气或峰流速计（用于测量肺部空气排出情况）。评估基线功能、血氧饱和度水平及供氧量和方法。

峰流速值分为三个区域：

绿色：基线的80%或更高（轻度发作）

黄色：基线的50%至80%之间（中度发作）

红色：<基线的50%（重度发作）

提醒：记录来电者对建议的反应、家庭护理指示及何时回电。

## 评估与行动

A. 是否存在以下任何情况？

●胸痛

●嘴唇或舌头发绀

●皮肤湿冷

●窒息感

●泡沫状粉红色或大量白色痰液

●意识改变

●突然发作的严重呼吸短促

●有肺栓塞、血栓或肺塌陷史

●使用吸入器未能缓解的哮喘病史和严重喘息

●无法说话

●流口水且无法吞咽

●吸入烟雾、火焰或气体后的呼吸困难

是 “呼叫救护车”

否 ↓ 转至B

B. 是否存在以下任何情况？

●因剧烈疼痛而难以深呼吸

●过去2小时内开始的严重气促、喘息或有声呼吸

●近期创伤、手术或分娩史

●吸入异物

●接触过之前引起显著反应的物质（如蜇伤、药物、植物、化学物质、食物或动物）

●断续说话

●无法平躺或需坐起来呼吸

●免疫抑制，年龄>60岁，有镰状细胞贫血或糖尿病病史，或长期卧床且体温>101°F (38.3°C)

●峰流速<基线的50%

●成人体温>103°F (39.4°C)

●呼吸短促逐渐加重

●治疗后持续喘息

●严重喘息或咳嗽，常用的雾化器或吸入器不可用

是 “立即寻求紧急医疗护理”

否 ↓ 转至C

C. 是否存在以下任何情况？

●断续说话

●紧绷的咳嗽

●休息时有轻微可听的喘息声

●呼吸时疼痛加剧

●上呼吸道感染且之前因相同症状住院

●因咳嗽或呼吸困难无法连续睡眠1-2小时以上

●有糖尿病或心脏病史

●峰流速为基线的50%至80%

是 “在2到4小时内寻求医疗护理”

否 ↓ 转至D

D. 是否存在以下任何情况？

●发烧

●体温>101°F (38.3°C)

●峰流速为基线的50%至80%

是 “在2天内病情未改善或恶化时报告给您的初级保健医生/诊所/急诊科”

否 ↓ 转至E

E. 是否存在以下任何情况？

●胸部不适持续不缓解或加重

●呼吸困难且无法缓解

●突然发作的严重症状，如胸痛、意识丧失等

是 “立即寻求紧急医疗护理或呼叫救护车”

否 “记录建议并鼓励患者病情恶化时回电或就诊初级保健医生”

如果来电者同意所给建议，记录通话并鼓励其在问题恶化时回电或就诊初级保健医生。如果来电者不同意所给建议，重新评估并向其建议跟进初级保健医生、诊所或急诊科。
