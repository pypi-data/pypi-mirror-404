# 过度换气

关键问题 姓名，年龄，发病时间，原因，用药情况，病史，伴随症状

其他需考虑的协议 焦虑 (36)；呼吸问题 (106)；胸痛 (123)；麻木和刺痛 (431)。

提醒：记录来电者对建议的反应、家庭护理指示及何时回电。

## 评估与行动

A. 是否有胸痛？

● 是 → 参见胸痛协议（123）

● 否 ↓ 转至 B

B. 除了呼吸急促外，是否还有以下任何症状？

● 嘴唇或指甲发绀

● 无法说话

● 必须坐起来才能呼吸

● 是 → 参见呼吸问题协议 (106)

● 否 ↓ 转至 C

C. 除了快速呼吸之外，是否还有以下任何症状？

● 严重疼痛

● 发热且对退烧措施无反应

● 异常困倦

● 是 → “在2到4小时内寻求医疗护理”

● 否 ↓ 转至 D

D. 除了深快呼吸外，是否有以下任何症状？

● 面部、口唇、手指或脚趾周围的麻木和刺痛感

● 无力

● 感觉要昏倒

● 手脚抽搐

● 焦虑或情绪激动

● 有类似症状的病史或先例

● 是 → “如果无改善，请回电或联系家庭医生预约”

并且

遵循家庭护理指导

● 否 遵循家庭护理指导


## 家庭护理指导

过度换气

● 坐下并专注于放慢呼吸，每5秒一次呼吸。

● 用纸袋盖住口鼻，深呼吸10次。

● 如果没有改善，继续使用袋子呼吸5到15分钟。

● 不用袋子休息几分钟。

● 反复进行带袋子和不带袋子的呼吸，直到状况改善。

## 如出现以下问题，请联系您的家庭医生/诊所/急诊室

● 无改善或状况恶化

● 发作期间突然发热

● 频繁发作


## 如果发生以下任何情况，请立即寻求急救护理

● 昏厥

● 抽搐

● 嘴唇或指甲发绀

● 胸痛

如果来电者同意所给建议，记录通话并鼓励其在问题恶化时回电或联系家庭医生。

如果来电者不同意所给建议，重新评估，并建议来电者与家庭医生、诊所或急诊室跟进。
