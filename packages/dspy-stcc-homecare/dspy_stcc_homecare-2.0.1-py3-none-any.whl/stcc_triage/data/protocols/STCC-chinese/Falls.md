# 跌倒

关键问题 姓名、年龄、发病时间、原因、跌落距离、用药情况、病史

其他需考虑的协议 酒精问题（21）；精神状态改变（28）；背部/颈部损伤（59）；瘀伤（109）；混乱（150）；糖尿病问题（187）；头晕（199）；家庭暴力（202）；四肢受伤（222）；头部受伤（311）；髋部疼痛、受伤（334）；颈部疼痛（415）；神经系统症状（418）；物质滥用或接触（582）；无力（649）。

护士提醒：每三位75岁以上居住在家的老年人中就有一位每年至少会发生一次严重的跌倒。在跌倒导致严重骨折后，有25%的人会在半年到一年内因行动不便或卧床不起而死亡。请参阅《家庭护理指导》（233）中的跌倒预防指南。

提醒：记录来电者对建议、家庭护理指导和何时回电的响应。

## 家庭护理指导

跌倒

●在受伤部位冰敷20分钟，每天4次，持续24至48小时。不要将冰直接放在皮肤上，使用湿布或其他织物作为隔离。

●预期会出现1周或更长时间的青紫斑痕。

●评估家中是否存在安全隐患并纠正问题。

●联系当地社区中心或老年人中心获取关于锻炼、跌倒预防计划和提高家庭安全性的建议。

●访问跌倒预防网站：www.cdc.gov/injury 或 www.stopfalls.org

跌倒预防指南

●定期进行锻炼以增强体力和平衡能力。

●每年进行一次眼科检查，并在需要时更换眼镜。

●请医生或药剂师审查药物，因为某些药物可能会改变视力、导致头晕或混乱。

●穿合脚的鞋子。

●固定或移除松动的地毯。

●清除杂物或可能导致绊倒的障碍物。

●在淋浴和浴缸地板上添加防滑安全贴纸。

●所有楼梯都安装扶手并确保照明充足。

●在浴室中加装扶手。

●在更衣区放置一个凳子或坐的地方——许多跌倒发生在尝试穿衣服时。

## 如出现以下问题，请联系您的主治医生/诊所/急诊

●症状持续或加重

●频繁发生跌倒

●行动能力逐渐困难

●出现剧烈疼痛



## 如果出现以下任何情况，立即寻求紧急医疗

●持续呕吐

●身体一侧无力
