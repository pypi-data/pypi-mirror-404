# 擦伤

关键问题：姓名，年龄，发病时间，原因，其他伤害，药物，疼痛评分，既往病史

需考虑的其他方案：异物（278）；裂伤（395）；刺伤（496）；皮肤病变：肿块、凸起和溃疡（556）；伤口愈合与感染（664）。

护士提醒：伤口被定义为清洁或污染。如果伤口受到污垢、粪便、唾液或土壤的污染；刺伤；撕裂伤；由飞溅物或压碎物造成，动物咬伤，烧伤或冻伤引起，则认为是污染伤口。

提示：记录求助者对建议的反应，家庭护理指导及何时回电。

## 评估与行动

A. 是否存在以下任何情况？

●难以控制出血

●有血友病史

●大面积身体受影响

是 “立即寻求紧急医疗”

否 ↓ 转到B

B. 是否存在以下任何情况？

●无法清除伤口中的污垢或其他异物

●源头污染，且最后一次破伤风疫苗接种已超过5年

是 “2至4小时内就医”

否 ↓ 转到C

C. 是否存在以下任何情况？

●有糖尿病史

●受影响部位活动困难

●伤口为24至48小时旧，并出现感染迹象：红肿、疼痛、触之发热、从创面延伸出的红线、渗液或脓液，或发热

是 “24小时内就医”

否 ↓ “如无改善，请回电或预约家庭医生”

遵循家庭护理指导



## 家庭护理指导

擦伤

●用干净的绷带或布直接按压伤口以控制出血。

●每天用肥皂和水彻底清洗伤口。

●在接下来几天内，每日2至3次涂抹抗生素软膏。遵循标签上的指示。

●1至2天内使用干燥、清洁的敷料覆盖伤口。

●每日检查伤口是否有感染迹象（红肿、疼痛、触之发热、从创面延伸出的红线、渗液或脓液，或发热）。

●如果敷料粘在伤口上，用水浸泡。

●如果伤口看起来湿润，请让伤口自然干燥5至10分钟，然后每天重新包扎以促进愈合。

## 如出现以下问题，请向您的家庭医生/诊所/急诊报告

●感染迹象

●愈合延迟 >1周

如果求助者同意所给建议，记录通话并鼓励求助者在情况恶化时回电或就诊于家庭医生。

如果求助者不同意所给建议，重新评估并建议求助者与家庭医生、诊所或急诊跟进。
