# 肩部疼痛/损伤

关键问题 姓名，年龄，发病时间，原因，过敏史，用药情况，病史，疼痛评分

## 评估与行动

A. 下列情况是否出现？

●肩部、颈部、下巴或胸部突然疼痛，并伴有呼吸短促或出汗

●受伤后肩部出现畸形、瘀伤和活动受限

●月经延迟超过4周的女性出现肩部和腹部突然疼痛

是 “呼叫救护车”

或

“立即寻求紧急医疗救助”

否 ↓ 转至B

B. 下列情况是否出现？

●最近有肩部、腹部或背部钝挫伤史

●发热，关节肿胀、红润或温暖，并伴有近期疾病

●无法抬起手臂或移动肩部

●腹痛放射到肩部

是 “2-4小时内寻求医疗护理”

否 ↓ 转至C

C. 下列情况是否出现？

●其他关节突然疼痛

●最近肩部受伤，且在冰敷、热疗和休息超过3天后疼痛未改善

是 “24小时内寻求医疗护理”

否 ↓ 转至D

D. 下列情况是否出现？

●肩部进行性酸痛，活动时加剧

●进行性疼痛和僵硬

●疼痛到一天结束或重复使用或侧向伸展手臂时加重

●腹腔镜手术后肩部或上胸部疼痛

是 “如无改善请回拨或预约家庭医生”

并

遵循家庭护理指导

否 遵循家庭护理指导



## 家庭护理指导

肩部疼痛/损伤

●初次受伤后，24至48小时内每次20到30分钟，每天4到6次使用冰敷。之后可使用湿热敷。休息肩膀24至48小时。不要直接将冰放在皮肤上。不要在加热垫上睡觉。

●避免提、拉或推重物。

●对于无损伤的疼痛，每次20到30分钟，每天4到6次使用湿热敷。

●如无已知损伤，可通过缓慢逐渐伸展来活动关节，尽量抬高手臂。向下、向前、向上及向两侧伸展。

●服用常规止痛药（阿司匹林、对乙酰氨基酚或布洛芬）缓解不适，并遵循标签上的指示。不要给儿童服用阿司匹林。肝病患者避免使用对乙酰氨基酚。

●如果在电脑或其他重复运动活动中工作，若肩部疼痛持续到一天结束时加重，可以考虑进行工作站的工效学评估。

## 如出现以下问题，请报告您的家庭医生/诊所/急诊科

●疼痛持续或加剧

●新的症状



## 如果出现下列任何情况，请立即寻求紧急医疗护理

●重复或突然的肩部、颈部、下巴或胸部疼痛，并伴有呼吸短促或出汗

如果呼叫者同意所给建议，记录通话并鼓励其在问题恶化时回拨或见家庭医生。

如果呼叫者不同意所给建议，重新评估并建议其与家庭医生、诊所或急诊科联系。
