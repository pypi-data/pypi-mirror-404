# 造口问题

关键问题：姓名、年龄、发病时间、原因、造口类型或造口袋原因、药物、病史

其他需要考虑的方案：腹痛，成人（9），儿童（13）；严重出血（90）；便秘（160）；伤口愈合和感染（664）。

护士提示：如果患者有造口并有疑虑或问题，请使用此方案。

提醒：记录咨询者的反应、家庭护理指示及何时回电。

## 评估与行动

A. 下列任何一项是否存在？

●造口处严重出血

●严重的腹痛或肿胀

●持续呕吐

●发热且腹部触痛或僵硬

●便秘，腹痛，肿胀和呕吐

是 “呼叫救护车” 或“立即寻求紧急医疗”

否 ↓ 转至 B

B. 下列任何一项是否存在？

●尿路造口且体温 >100.4°F (38°C)，且尿液呈粉红色、红色或异常浑浊

●恶心，食欲不振，腹痛

●引流液渗入手术伤口

●服用抗凝药物且新出现的造口出血或尿液呈粉红或红色

●无造口排泄物

●大便带血

是 “2至4小时内寻求医疗护理”

否 ↓ 转至 C

C. 下列任何一项是否存在？

●超过4天未有排便且便秘对家庭护理措施无反应

●无法管理造口或造口袋伤口

是 “24小时内寻求医疗护理”

否 ↓ 转至 D

D. 下列任何一项是否存在？

●造口处少量出血

●新出现的瘙痒和灼烧感在造口或造口袋部位

●造口排泄物减少

●轻度便秘

●新服用药物且正常肠道习惯改变

是 “如无改善，请回电或预约初级保健医生”并遵循家庭护理指示

否 遵循家庭护理指示



## 家庭护理指示

造口问题

造口护理

●检查器具，确保各部分紧密连接。

●如果造口有渗漏，移除器具，用水清洁皮肤，彻底干燥，并按护士或初级保健医生的指导准备皮肤；应用新的器具。保持5分钟以确保袋子牢固贴合于皮肤上。

●按指示服用处方药物。

●当回肠造口袋充盈1/3-1/2时排空。

●肥皂可能干扰皮肤屏障粘合剂的作用；在使用肥皂清洁造口周围区域前，请与ET护士或提供者讨论。

●定期更换皮肤屏障—通常每3至4天一次，或按提供者的指示进行。

●在应用新的皮肤屏障和器具之前，确保造口周围的皮肤干净干燥。

出血的造口

●记住少量出血是正常的，组织容易出血，类似于刷牙后牙龈出血的情况。

便秘

●确保饮食量充足、富含体积（高纤维）及液体（每天6到8杯水，除非在限流饮食中）。

●每天早晨喝一杯热饮料，如咖啡、茶、柠檬热水或李子汁。

●按照指示使用大便软化剂、泻药和灌肠。

●当服用含可待因或其他增加便秘的药物时，请遵循初级保健医生开具的便秘预防计划。

额外指示








## 请向您的初级保健医生/诊所/急诊科报告以下问题

●发热或出血持续或加剧

●家庭护理措施后便秘持续或加重

●引流液渗入手术伤口

●家庭护理措施后仍有持续渗漏

●使用处方药物后瘙痒和灼烧感持续

●大便带血

●持续腹泻

●造口周围皮肤新出现的肿块

●持续的皮肤刺激对家庭护理措施无反应。



## 如果发生以下任何情况，请立即寻求紧急医疗援助

●严重出血

●严重的腹痛或肿胀

●持续呕吐

●发热且腹部触痛和僵硬

如果咨询者同意所给建议，记录通话并鼓励在问题恶化时回电或就诊初级保健医生。

如果咨询者不同意所给建议，重新评估，并建议与初级保健医生、诊所或急诊科跟进。
