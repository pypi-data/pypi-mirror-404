# 胸痛

关键问题：姓名、年龄、发病时间、既往心肌梗死史、冠状动脉疾病史、糖尿病史、肺栓塞史、深静脉血栓形成史、伴随症状、用药情况、疼痛评分、病史、胸部植入装置

考虑的其他协议：焦虑（36）；呼吸问题（106）；普通感冒症状（146）；充血（153）；心力衰竭（157）；咳嗽（170）；头晕（199）；胃灼热（316）；心率问题（320）；消化不良（368）；成人呕吐（642）；过度出汗（594）；无力（649）。

护士提示：有许多状况可引起胸痛，其中一些可能是潜在的生命威胁。在对出现胸痛的患者进行分诊时，请务必谨慎。附录S（712）《胸痛的原因和特征》提供给护士以帮助其更好地了解引起胸痛的各种情况。但不要用于尝试诊断患者的状况。

提醒：记录患者对建议、家庭护理指导及何时回拨的响应。

## 评估与行动

A. 是否存在以下任何情况？

●持续或间歇性疼痛、紧绷感、压迫感或不适伴随有：

●呼吸短促

●头晕或无力

●皮肤湿冷

●恶心或呕吐

●颈部、肩部、下颚、背部或手臂的疼痛

●脸部、嘴唇、耳垂或指甲呈蓝色或灰色

●心悸或颤动

●胸痛持续存在，休息、止痛药、抗酸剂或每隔5分钟服用一次硝酸甘油（连续3次）均不能缓解

是 “呼叫救护车”

并

“咀嚼一片成人阿司匹林（除非对阿司匹林过敏）”

否 转至B

B. 是否存在以下任何情况？

●近期有创伤，且疼痛随活动加剧。将热敷袋放在受影响区域，每天4次，每次20分钟。

●咳嗽持续或加重，采取上述措施后无改善。

是 “采取相应措施并记录症状”

否 转至C

C. 是否存在以下任何情况？

●没有明显的创伤或其他原因，且疼痛在休息时减轻但在活动时加剧。服用硝酸甘油（按医生指示）；如果3-5分钟后仍无缓解，再服用一次，并有人陪同前往急诊室或呼叫救护车。

是 “记录症状并建议立即就医”

否 转至D

D. 是否存在以下任何情况？

●疼痛与已知的创伤或其他原因相符，且采取上述措施后有所改善。继续观察症状并在需要时重复使用热敷或止痛药。

是 “记录症状并建议随访医生”

否 “转诊给初级保健提供者进行进一步评估和治疗”

## 如下问题请告知您的主治医师/诊所/急诊室

●状况没有改善或恶化

●局部有疼痛性水泡或皮疹的区域

## 如果出现以下任何情况，请立即寻求紧急护理

●持续或间歇性的疼痛、紧绷感、压迫感或不适伴随：

●呼吸短促

●头晕

●皮肤湿冷

●恶心或呕吐

●颈部、肩部、下颚、牙齿、背部或手臂的疼痛

●脸部、嘴唇、耳垂或指甲呈蓝色或灰色

●心悸或颤动

●连续两次每隔5分钟服用一次硝酸甘油后仍无缓解

●植入体内除颤器反复放电

●剧烈疼痛

如果患者同意所给建议，记录此次咨询，并鼓励患者如状况恶化时回拨电话或就诊于初级保健提供者。

如果患者不同意所给建议，请重新评估并建议其与主治医师、诊所或急诊室跟进。
