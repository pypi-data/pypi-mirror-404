# 纹身问题

关键问题 姓名，年龄，最近纹身的日期，纹身位置，药物使用情况，病史，疼痛评分，破伤风免疫状态

其他需要考虑的方案 擦伤（19）；割伤（395）；穿刺问题（445）；伤口愈合和感染（664）。

## 家庭护理指导

纹身问题

●通过直接对受伤部位施压来止血。

●使用冷敷包以减少肿胀、瘀伤或瘙痒。不要将冰块直接放在皮肤上；在冰和皮肤之间放置一块布作为屏障。

●24小时后移除绷带，并涂抹抗生素软膏（三重抗生素软膏或新霉素）。如果伤口变脏或因衣物摩擦而发炎，可以用无粘连的绷带更换。每天用干净的绷带更换一次，如果绷带湿了也要更换。

●如果绷带粘在纹身上，将其放在流动水下并轻轻移除。

●保持皮肤清洁。不要擦洗。避免使用强效肥皂、洗涤剂或其他化学物质。清洁后轻轻拍干。在愈合期间每天检查纹身是否有感染迹象。

●一些清澈的黄色或血色的引流是正常的。报告感染迹象：发红、肿胀、恶臭的引流、红色条纹、触感发热，或疼痛加剧。

●尽可能让纹身暴露在空气中进行愈合。

●每天涂抹几次保湿霜并避免日晒。

●14天内避免使用热水浴缸或游泳池。

●如果出现荨麻疹或瘙痒，请服用抗组胺药（如苯海拉明或扑尔敏）。请按照标签上的指示服用。

●服用常规止痛药。不要给儿童服用阿司匹林。年龄<20岁者避免使用类似阿司匹林的产品。有肝病的患者避免使用对乙酰氨基酚。有肾病、胃部问题的患者或怀孕的情况下，应避免使用布洛芬。请按照标签上的指示服用。

●愈合可能需要长达两周的时间。

●如果您计划进行MRI检查，请告知放射科医师或技术人员您有纹身，并指出纹身的位置和颜色。某些纹身可能会扭曲成像或对MRI产生反应。

其他指导



## 如果出现以下问题，请向您的家庭医生/诊所/急诊室报告

●头痛、肌肉疼痛、全身不适感或发热

●情况没有改善或恶化

●体温>100.4°F（38°C）

●感染迹象



## 如发生以下任何情况，立即寻求紧急医疗帮助

●纹身下的区域肿胀、冰冷、潮湿或疼痛

●靠近纹身处的关节迅速肿胀

●疼痛部位，肌肉痉挛，下巴和颈部僵硬，或吞咽困难或张口困难

如果来电者不同意给出的建议，请重新评估并建议来电者与家庭医生（PCP）、诊所或急诊室联系。
