# 石膏/夹板问题

关键问题 姓名，年龄，起始时间，原因，石膏或夹板类型，位置和应用日期，受伤或手术类型，病史，药物

●新发呼吸困难

B. 下列任何一项是否出现？

●严重的疼痛、肿胀或紧绷感，即使抬高患肢或采取家庭护理措施后仍未缓解

●感染的迹象：疼痛、肿胀、渗液、发热或伤口周围有红线

●手指或脚趾麻木或刺痛，即使抬高患肢或采取家庭护理措施后仍未缓解

否 ↓ 转至 C

C. 下列任何一项是否出现？

●通过家庭护理措施，肿胀、疼痛或紧绷感有所缓解

●石膏/夹板裂开或不稳定

●新发手指或脚趾不能活动

是 “24小时内寻求医疗救助”

否 ↓ 转至 D

D. 下列任何一项是否出现？

●石膏潮湿

●通过家庭护理措施，持续的肿胀或紧绷感有所改善

●感觉石膏过松或过紧

## 家庭护理指导

石膏/夹板问题

●如果感到石膏过紧，将受伤部位抬高至心脏以上。

●如果感到夹板过紧，松开绷带，抬高患肢，并在该区域每隔2小时使用冰袋冷敷20分钟，持续24至48小时。不要直接将冰放在皮肤上；应在冰和皮肤之间放置一条湿布或其他织物屏障。

●如果石膏或夹板损坏，请用宽的胶带或弹性绷带提供支撑，并在次日预约PCP进行修复。

●如果石膏或夹板潮湿，用毛巾擦干并根据需要使用低档位吹风机吹干。

●如果有瘙痒感，可以轻轻撒上一层滑石粉或使用设置为冷风的吹风机。不要将任何物品插入石膏中。刮伤的皮肤可能会感染。

## 如出现以下问题，请报告给您的PCP/诊所/急诊室

●石膏或夹板不稳定或损坏

●新发的石膏内疼痛

●没有改善或疼痛加剧

●感染迹象



## 如果出现下列任何情况，立即寻求紧急医疗救助

●血液循环问题的症状：手指或脚趾变蓝、冷或麻木

●严重的疼痛或肿胀，即使抬高患肢或采取家庭护理措施后仍未缓解

●呼吸困难
