# 蛲虫

关键问题：姓名，年龄，发病时间，已知或疑似蛲虫感染，用药情况，病史

其他需考虑的方案：床虱暴露或担忧（67）；瘙痒（382）；直肠问题（513）。

提醒：记录来电者对建议、家庭护理指导以及何时回拨的反应。

## 评估与行动

A. 是否存在以下任何情况？

●直肠区域感染迹象（疼痛，肿胀，发红，分泌物，发热）

●夜间及凌晨时分严重的直肠瘙痒加剧

●直肠或阴道区域出现1/4至1/2英寸长的白色线状虫体

●粪便中可见虫体

是 “请在24小时内寻求医疗护理”

否 ↓ 转到B

B. 是否存在以下任何情况？

●直肠区域轻微发红、瘙痒或触痛

●儿童睡眠困难，烦躁不安或阴道刺激

●接触过蛲虫感染儿童的床上用品或床单

●治疗1周后直肠症状持续不缓解

●家庭成员被诊断为蛲虫感染并担心传播问题

是 “如果没有改善，请回拨电话或预约主治医生” 并遵循家庭护理指导

否 遵循家庭护理指导



## 家庭护理指导

蛲虫

●为了检测儿童是否患有蛲虫，在睡前几小时在暗室内用灯光照射儿童的肛门。如果存在蛲虫，它们会重新爬回肛门。

●修剪指甲，并鼓励良好的洗手习惯。

●避免咬指甲或吮吸拇指。

●用热水和肥皂水清洗衣物及内衣直至蛲虫消失。

●治疗后2周内每天真空清洁或拖扫卧室。

●每天早上洗澡并清洁受影响区域。淋浴为佳。

●穿短裤或内裤在睡衣下面。

●为了减轻瘙痒：

●将氧化锌或1%氢化可的松软膏涂抹于受影响区域。

●用温水加泻盐或食盐泡澡。

●可能需要处方药来消除蛲虫。按指示完整服用药物。

●尝试非处方蛲虫药物（如Reese’s），并按照标签说明使用。
