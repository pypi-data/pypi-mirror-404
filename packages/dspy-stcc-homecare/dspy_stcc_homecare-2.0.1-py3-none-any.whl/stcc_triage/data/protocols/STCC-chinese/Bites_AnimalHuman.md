# 咬伤，动物/人类

关键问题：姓名、年龄、起始时间、原因、咬伤部位、破伤风状态、动物类型、动物的免疫状态、药物使用情况、疼痛评分、病史

其他需要考虑的方案：出血严重（90）；血液/体液暴露（93）；家庭暴力（202）；HIV暴露（340）；破伤风免疫接种（355）；撕裂伤（395）；刺伤（496）；伤口愈合和感染（664）。

护士提醒：大型犬可以造成最严重的伤口，导致类似压迫性损伤的情况，损害血管、肌腱、肌肉、神经和骨骼。猫咬伤因尖锐牙齿造成的刺伤而具有高度感染风险，将细菌深深推入组织中。手部被咬伤由于手部许多结构的血液供应相对差，因此感染率最高。局部感染和蜂窝织炎是导致咬伤伤口发病率的主要原因，并且可能特别在免疫功能低下个体中导致败血症。无明显诱因来自野生动物或看似患病动物（狗、猫、臭鼬、蝙蝠和浣熊）的咬伤引发狂犬病暴露的担忧。

提醒：记录来电者对建议、家庭护理指导及何时回拨的反应。

## 家庭护理指导

动物/人类咬伤

●每天用肥皂布清洁受伤区域，然后用水彻底冲洗。

●按照标签说明，每日3次涂抹常规使用的抗生素软膏（例如：Mycitracin Triple Antibiotic [bacitracin, neomycin, and polymyxin B]、Neosporin、Polysporin）。如有需要，请向药剂师咨询其他产品建议。

●用干净干燥的敷料覆盖伤口2天，之后让伤口暴露于空气中，除非有血液渗出。

●如果敷料粘在伤口上，请用水冲洗。

●如果伤口看起来湿润，让伤口自然干燥5到10分钟，然后重新包扎以促进愈合。

●受伤后前24小时内使用冰袋减轻肿胀。24小时后对受伤部位进行热敷。不要将冰直接放在皮肤上；在冰和皮肤之间放置一层布料作为屏障。

●每日检查伤口是否有感染迹象。猫咬伤和人咬伤容易发生感染。

●观察动物两周，注意是否出现狂犬病或疾病症状。

●向动物管理部门或相关机构报告动物咬伤情况。

●报告蝙蝠和臭鼬的咬伤。

●在下列情况下报告狗和猫的咬伤：

●动物患病

●无明显诱因的咬伤

●流浪动物

●没有接种狂犬疫苗的迹象

●受伤环境可疑或不清楚/不确定
