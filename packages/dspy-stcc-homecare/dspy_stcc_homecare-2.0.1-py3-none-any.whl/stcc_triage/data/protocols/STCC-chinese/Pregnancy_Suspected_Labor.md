# 妊娠，疑似临产

关键问题：姓名、年龄、起始时间、原因、孕周、妊娠次数、用药情况、既往史

其他需考虑的方案：腹痛（9）；妊娠，胎动异常（468）；妊娠，阴道流液（474）；妊娠并发症（481）；妊娠，疑似早产<36周（488）；妊娠，排尿问题（491）；妊娠，阴道出血（493）

提醒：记录来电者对建议的反应、居家护理指导以及何时回电。

## 评估与行动

A. 孕期超过20周，并且存在以下任何情况吗？

●即将分娩，胎儿头部已露出

●大量阴道出血

●抽搐或惊厥

●剧烈腹痛

●羊膜破裂伴脐带脱垂

是 “呼叫救护车” 或 “立即寻求紧急医疗”

否 ↓ 转至 B

B. 存在以下任何情况吗？

●严重头痛

●已知胎儿为臀位或横位，在首次出现宫缩或流液时

●流出棕色或绿色阴道液体

●当前有疱疹发作，并伴有规律且强烈的宫缩

●宫缩和孕周<37周

●无胎动

是 “立即寻求医疗”

否 ↓ 转至 C

C. 存在以下任何情况吗？

●第一次怀孕，强烈且规律的宫缩（每5分钟一次，持续45至60秒，持续1到2小时）

●非首次怀孕，强烈且规律的宫缩（每5到10分钟一次，持续45至60秒，持续1到2小时）

●流出清澈阴道液体

●已知有高血压、糖尿病或其他妊娠并发症史

●快速分娩史

●前往分娩机构路程较长

●1小时内少于10次胎动

●宫缩强度或频率增加

●骨盆压力

是 “2小时内寻求医疗”

否 ↓ 转至 D

D. 存在以下任何情况吗？

●不规律的宫缩

●点状出血

●背部、脚部或腹股沟疼痛

●排尿时有急迫感、频次增加或疼痛

是 “无改善请回电或联系主治医生或产科医生预约” 并遵循居家护理指导

否 遵循居家护理指导



## 居家护理指导

妊娠，疑似临产

●散步、淋浴、按摩、背部使用热敷或冷敷以缓解不适，改变体位或进行骨盆摇摆运动帮助早期分娩。

●吃清淡的食物。

●监测宫缩：频率 = 从一次宫缩的开始到下一次宫缩的开始（即5分钟）；持续时间 = 宫缩持续的时间（即30至45秒）。

●监测胎动；如果1小时内少于10次胎动，请寻求医疗帮助。

●若阴道流液且脐带脱垂：

●将电话交给他人拨打急救电话

●采取膝胸位，头部向下，臀部向上

●增加液体摄入量；避免咖啡因。如有宫缩，喝32盎司的水，左侧卧位，并监测宫缩。如果1小时内宫缩超过4次，请回电。保持膀胱排空。
