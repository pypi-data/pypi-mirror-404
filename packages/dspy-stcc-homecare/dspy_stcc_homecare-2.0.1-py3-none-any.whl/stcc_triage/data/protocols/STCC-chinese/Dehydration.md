# 脱水

关键问题 姓名、年龄、发病时间、原因、药物、病史、来电者或分诊人员对可能的脱水症状感到担忧

其他考虑协议 糖尿病问题（187）；成人腹泻（192），儿童腹泻（195）；成人发热（250），儿童发热（253）；热暴露问题（323）；成人呕吐（642），儿童呕吐（645）；多汗症（594）。

护士提示：脱水可能迅速成为需要及时医疗干预的严重紧急情况。持续或频繁的脱水可能导致尿路感染、肾结石和肾衰竭。常见原因包括液体摄入不足、腹泻、过度出汗、糖尿病和痴呆等疾病，以及利尿剂等药物的影响。

●中度或重度脱水的症状

●排尿不频繁（>12小时内<1次）

●尿液呈深黄色或琥珀色

●极度口渴

●非常干燥的口腔、嘴唇和眼睛

●哭泣无泪

●头晕导致站立或行走困难

●心跳快速或微弱

●轻度脱水的症状

●排尿不频繁（>24小时内<3至4次）

●尿液呈深黄色

●极度口渴

●无力或异常疲乏

●干燥的口腔、舌头，粘稠的唾液

提醒：记录来电者对建议的反应、家庭护理指导以及何时回电。

## 评估与行动

A. 下列几项是否出现？

●精神状态改变：倦怠、异常易怒、混乱、谵妄或难以唤醒

●严重头痛、颈部僵硬或前屈头部时疼痛

●呕吐鲜红色血液或类似深咖啡色的呕吐物

●血便或黑便

●严重腹痛或胸痛

●呼吸困难

●发热>103°F（39.4°C）且>12小时内无尿

●呼吸急促或快速，或者儿童或老年人显得非常病态

●儿童或老年人的脱水迹象：

●排尿不频繁（>12小时内<1次）

●尿液呈深黄色

●眼睛凹陷或前囟门凹陷

●捏起的皮肤无法迅速回弹

●极度口渴

●口腔或黏膜干燥

●婴儿哭泣无泪

是 “立即寻求紧急护理”

否 ↓ 转至B

B. 下列几项是否出现？

●儿童4至6小时内无尿

●成人的脱水迹象：

●排尿不频繁（>12小时内<1次）

●尿液呈深黄色

●眼睛凹陷或前囟门凹陷

●捏起的皮肤无法迅速回弹

●极度口渴

●口腔或黏膜干燥

●持续呕吐、腹泻或发热超过24小时

●糖尿病并伴有持续呕吐、腹泻或发热

●年龄>60岁，卧床不起，或者免疫系统受损

●头晕、头重脚轻、行走不稳或站立不稳

是 “在2至4小时内寻求医疗护理”

否 ↓ 转至C

C. 下列任何一项是否出现？

●头晕、头重脚轻、行走不稳或站立不稳

是 “立即寻求紧急护理”

否 “请记录并鼓励来电者如果情况恶化，请回电或咨询家庭医生（PCP）。”

## 向您的家庭医生/诊所/急诊科报告以下问题

●高热、无力或腹痛超过2小时

●24小时内无改善或状况恶化

●成人持续的脱水迹象

●头晕、头重脚轻、行走不稳或站立不稳

## 如果发生以下任何情况，请立即寻求紧急护理

●精神状态改变

●呕吐鲜红色血液或类似深咖啡色的呕吐物

●血便或黑便

●严重头痛、颈部僵硬或前屈头部时疼痛

●呼吸困难

●发热>103°F（39.4°C）

●老年人或儿童持续的脱水迹象

如果来电者同意所给建议，请记录通话并鼓励其情况恶化时回电或咨询家庭医生。

如果来电者不同意所给建议，重新评估并建议其联系家庭医生、诊所或急诊科。
