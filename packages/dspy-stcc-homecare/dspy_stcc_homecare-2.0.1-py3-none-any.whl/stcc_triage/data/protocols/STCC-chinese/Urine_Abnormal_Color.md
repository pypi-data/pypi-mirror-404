# 尿液，颜色异常

关键问题 姓名、年龄、发病时间、用药情况、病史

其他考虑的方案 腹痛（成人9）、腹痛（儿童13）；脱水（180）；肝炎（329）；妊娠期排尿问题（491）；导尿管/肾造瘘管问题（621）；排尿疼痛（628）

提醒：记录呼叫者对建议的反应、家庭护理指示以及何时回电。

## 评估与行动

A. 下列任何情况是否存在？

●粉色、红色或烟棕色尿液，且近期有背部或腹部创伤或剧烈疼痛

●发热和腰部或背部疼痛

●存在泌尿支架且伴有剧烈疼痛或大量鲜红血液

是 “请在2至4小时内寻求医疗护理”

否 ↓ 转B

B. 下列任何情况是否存在？

●近期出现深棕色尿液、大便颜色浅白，以及皮肤和眼睛发黄

●尿中带血，无论有无疼痛

是 “请在24小时内寻求医疗护理”

否 ↓ 转C

C. 下列任何情况是否存在？

●深黄色或橙色尿液，并且存在以下一种或多种症状：

●恶心

●呕吐

●腹泻

●发热

●暴露于炎热气候

●减少液体摄入量

●尿量减少

●使用可导致尿液颜色变化的药物：苯唑西平盐酸盐（Pyridium）、苯妥英钠（Dilantin）、酚酞、甲硝唑（Flagyl）或呋喃妥因（Furadantin）

●近期摄入过红菜头、甜菜、黑莓或其他红色食物

●新的处方药

是 “如果情况没有改善，请回电或预约初级保健医生”

并且

遵循家庭护理指示

否 遵循家庭护理指示



## 家庭护理指示

尿液，颜色异常

●增加液体摄入量。

●避免可能引起尿色变化的食物（红菜头、甜菜、黑莓或其他红色食物），并观察尿色是否有变化。尿色应在24小时内恢复到正常颜色。

●如果目前正在服用苯唑西平盐酸盐（Pyridium）治疗泌尿道感染，预计尿液会呈现红色或橙色。

●蓝色或绿色尿液通常与食物或药物摄入有关。

## 向您的初级保健医生/诊所/急诊科报告下列问题

●发热、排尿疼痛或背部疼痛

●家庭护理措施后情况持续超过24小时或恶化

●服用新药后出现意外的颜色变化

●持续恶心、呕吐、腹泻和减少液体摄入量



## 如果发生以下任何情况，请立即寻求紧急医疗援助

●创伤性撞击背部、侧部或腹部，或跌倒或事故后的肉眼可见的血尿并伴有腹痛加剧

如果呼叫者同意所给建议，请记录电话，并鼓励呼叫者在问题恶化时回电或看初级保健医生。

如果呼叫者不同意所给建议，请重新评估并建议呼叫者与初级保健医生、诊所或急诊科联系。
