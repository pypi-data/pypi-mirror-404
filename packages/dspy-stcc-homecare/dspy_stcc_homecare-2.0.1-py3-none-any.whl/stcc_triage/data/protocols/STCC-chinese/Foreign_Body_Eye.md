# 眼内异物

关键问题：姓名，年龄，发病时间，原因，用药情况，病史

其他需要考虑的方案：眼部损伤（225）；眼科问题（228）；视力问题（639）。

提醒：记录来电者对建议、家庭护理指导和何时回拨的反应。

## 评估与行动

A. 下列任何一项是否存在？

●物体嵌入眼球

●冲洗眼睛内的化学物质后仍有剧烈疼痛

●突然出现视力变化

●眼球突出

●虹膜部分有血液

●受伤眼有透明胶状物流出

●移除异物后仍感剧痛

●瞳孔大小不等

是 “立即寻求紧急医疗”

否 ↓ 转至 B

B. 下列任何一项是否存在？

●异物覆盖在虹膜上

●肿胀、疼痛或流泪持续超过30分钟

●因热水、化学物质或异物造成伤害，家庭护理后仍感疼痛

●无法移除自由浮动的异物

是 “2到4小时内寻求医疗”

并且

遵循家庭护理指导

否 ↓ 转至 C

C. 下列任何一项是否存在？

●受伤或移除异物后24小时仍有不适或刺激感

●伤口感染迹象：疼痛、肿胀、发红、流液或发烧

●无法取下隐形眼镜

是 “24小时内寻求医疗”

否 “若无改善，请回拨或预约家庭医生”

并且

遵循家庭护理指导



## 家庭护理指导

眼内异物

●眼睛中的化学物质：立即用冷水冲洗眼睛20至30分钟。将头倾斜，使受伤的眼睛朝下，并置于流水下。分开上下眼睑，让水流过眼球内部到外部。

●不要尝试移除：

●嵌入眼内的物体

●金属碎片

●覆盖在虹膜上的异物

●移除异物（如绒毛、尘埃颗粒或睫毛）的方法：

●向下拉低眼皮，并用湿润的手帕、纸巾或棉签的一角轻轻取出异物。

●将上眼皮翻过下眼皮并保持片刻，放开后检查是否能看到异物；如果可以，则将其取出。

●不要揉眼睛。

●应用冰袋或冷敷以减轻不适。

●服用常规止痛药（如阿司匹林、对乙酰氨基酚或布洛芬）。不要给儿童服用阿司匹林。20岁以下避免使用类似阿司匹林的产品。肝病患者避免使用对乙酰氨基酚。肾病、胃部问题患者或孕妇应避免使用布洛芬。请遵循标签上的指示。

## 出现下列任何问题时向您的家庭医生/诊所/急诊报告

●疼痛、肿胀、流液或发烧加剧

●视力变化

●48小时后疼痛无改善

●无法移除自由浮动的异物
