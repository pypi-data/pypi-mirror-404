# 婴儿溢奶

关键问题：姓名、年龄、起始时间、用药情况、病史

其他需要考虑的协议：母乳喂养问题（98）；脱水（180）；儿童呕吐（645）。

提醒：记录来电者对建议的回应、家庭护理指导以及何时回拨。

## 评估与行动

A. 下列任何症状是否出现？

●呼吸困难

●面部、嘴唇、指甲或耳垂呈蓝色或灰色

●反应迟钝

是 “呼叫救护车”

或

“立即寻求紧急医疗援助”

否 ↓ 继续B

B. 下列任何症状是否出现？

●持续呕吐

●之后有窒息或咳嗽

●吐出物中有血液或深绿色胆汁

●体温 >100.4°F (38°C) 且年龄 <3 个月

●脱水迹象：

●尿量减少

●眼窝凹陷

●皮肤弹性差（捏起后不能迅速恢复）

●极度口渴或口腔干燥

●哭时无泪

●前囟门凹陷

●新生儿无排便

●喷射性呕吐

是 “2小时内就医”

否 ↓ 继续C

C. 下列任何症状是否出现？

●持续烦躁

●腹泻

●家长担心婴儿体重增长不足

是 “24小时内就医”

否 ↓ 续D

D. 下列任何症状是否出现？

●体温升高且年龄 >3 个月

●婴儿摄入超过1到2口食物后频繁溢奶

●体重减轻

●便秘加剧

●持续溢奶

●家长担忧

是 “如果没有改善，请回拨或预约儿科医生”

并

遵循家庭护理指导

否 遵循家庭护理指导



## 家庭护理指导

婴儿溢奶

●请记住，喂食后不久非剧烈地吐出少量胃内容物是婴儿的常见情况。婴儿开始坐立后应该会自然好转。

●在喂养期间多次拍嗝。

●给予小而频繁的喂养，避免过度喂养或喂养过快。

●喂食后让婴儿处于直立位置，如放在摇篮或婴儿背带中。避免喂养后紧抱或晃动婴儿。

●抬高床头并使婴儿侧卧。

●更换尿布时避免使用过紧的尿布或束缚婴儿。

●如果进行母乳喂养，尝试每次只用一侧乳房哺乳，并泵出另一侧的奶。避免使用安抚奶嘴。

## 如果出现以下问题，请联系您的儿科医生/诊所/急诊

●家庭护理措施无效或情况恶化

●体重减轻或未能正常增重

●便秘加剧

●持续溢奶

●吐出物中有血液或深绿色胆汁

●脱水迹象



## 如发生下列任何症状，立即寻求紧急医疗援助

●呼吸困难

●反应迟钝

●面部、嘴唇、指甲或耳垂呈蓝色或灰色
