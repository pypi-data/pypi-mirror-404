# 寨卡病毒

关键问题 姓名，年龄，发病时间，病因，药物使用情况，病史，怀孕状态，近期是否前往寨卡病毒传播地区，最近是否有蚊虫叮咬

其他需考虑的方案 意识改变 (28)；成人发热 (250)；儿童发热 (253)；头痛 (308)；麻木和刺痛 (431)；成人皮疹 (500)；儿童皮疹 (505)；无力 (649)。

护士提醒：仅当诊断为寨卡病毒或来电者对寨卡病毒有疑虑且出现症状（蚊虫叮咬后 3 至 14 天内）时，且已知前往过寨卡病毒感染区域或通过无保护的性行为暴露于感染者的环境中时使用此方案。目前尚无针对寨卡病毒的治疗方法，治疗主要是对症处理。

提醒：记录来电者对建议的反应、家庭护理指导以及何时回电。

## 家庭护理指导

寨卡病毒

●减少感染寨卡病毒的风险：

●避免接触感染者的体液、尿液或粪便，尤其是在怀孕期间。如果无法避免接触，使用手套和漂白剂。

●涂抹驱蚊剂。不要对 2 个月以下的婴儿使用驱蚊剂。可以使用蚊帐覆盖 2 个月以下婴儿的推车、童车或摇篮以保护他们免受蚊虫叮咬。不要给 3 岁以下儿童使用含有柠檬桉油或para-menthane-diol的产品。不要将驱蚊剂涂抹在孩子的手、眼睛、嘴巴以及任何割伤或刺激皮肤上。如果同时使用防晒霜，请先涂防晒霜再涂驱蚊剂。

●穿长袖衬衫、长裤和袜子。

●尽量减少裸露的皮肤面积。

●穿着浅色衣物。

●用纱窗覆盖敞开的门窗，并修复或更换有孔洞或不合适的纱窗。

●避免积水，排干你房产上的物体或区域内的积水。

●清理堵塞的雨水槽。

●避免与已知感染寨卡病毒或疑似感染者进行性行为。使用避孕套（男性或女性避孕套）。

●休息。

●多喝水。

●服用常规止痛药缓解不适。避免使用阿司匹林和类似产品（非甾体抗炎药）。如果存在肝病，避免使用对乙酰氨基酚。遵循药品标签上的说明。使用随药附带的剂量器、测量工具或药店提供的药物注射器。家庭茶匙往往无法准确量取药物。

●有关寨卡旅行信息和受影响地区的最新信息，请参阅 CDC 网站：cdc.gov/zika 或 wwwnc.cdc.gov/travel/page/zika-travel-information

## 如果出现以下任何情况，请立即寻求紧急护理

●呼吸困难

●瘫痪

●高热

如果来电者同意所给建议，记录通话并鼓励来电者在问题恶化时回电或就诊医生。

如果来电者不同意所给建议，重新评估并建议来电者与医生、诊所或急诊联系。
