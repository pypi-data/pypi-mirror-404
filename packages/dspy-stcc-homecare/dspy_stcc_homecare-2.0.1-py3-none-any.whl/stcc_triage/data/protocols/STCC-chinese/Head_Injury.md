# 头部损伤

关键问题 姓名、年龄、发病时间、原因、过敏史、受伤机制、药物使用情况、相关症状、病史

其他需考虑的协议 意识改变（28）；背部/颈部损伤（59）；混乱（150）；跌倒（240）；头痛（308）；成人呕吐（642），儿童呕吐（645）。

护士提示：

●头部受伤时，应始终考虑颈部受伤。评估是否存在无力、不协调、麻木或颈痛的情况。

●意识改变可能是创伤后头部损伤的最早迹象之一，尤其是在儿童或老年人中。

提醒：记录来电者对建议的反应、家庭护理指导以及何时回电。

## 家庭护理指导

头部损伤

●每2小时使用冰袋或冷敷20分钟以减轻肿胀和不适。在皮肤与冰之间放置布障。

●允许儿童受伤后睡眠。24小时内每2小时唤醒一次，评估其警觉性和反应性。

●受伤后的头24小时内避免剧烈活动。在安静的环境中休息，头部略微抬高。

●使用对乙酰氨基酚缓解不适。不要给儿童服用阿司匹林。如果年龄<20岁，避免类似阿司匹林的产品。如果有肝病，请避免使用对乙酰氨基酚。如果有肾病、胃部问题或怀孕的情况下，避免使用布洛芬。请遵循标签上的说明。

●避免使用阿司匹林和布洛芬。

●受伤后的头24小时内避免饮酒、安眠药或镇静剂。

●避免大量饮食。

额外指示



## 如出现以下问题，请联系您的家庭医生/诊所/急诊室

●状况未改善或恶化

●持续头痛

●冰敷后超过24小时的持续肿胀

●鼻孔或耳朵流出血液或清澈液体



## 如果发生以下任何情况，立即寻求紧急医疗护理

●混乱、方向感丧失、激动或视力改变

●意识改变或难以唤醒

●手臂或腿部麻木、刺痛或无力

●持续呕吐、严重头痛、言语问题、抽搐或嗜睡

●儿童不认识父母或看护人

●持续的麻木和刺痛

如果来电者同意给出的建议，记录通话并鼓励来电者在情况恶化时回电或联系家庭医生。

如果来电者不同意给出的建议，重新评估，并建议来电者与家庭医生、诊所或急诊室进行跟进。
