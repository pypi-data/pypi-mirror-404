# 大便失禁

关键问题：姓名，年龄，起病时间，过敏史，用药情况，既往史

其他需考虑的方案：成人腹痛（9），儿童腹痛（13）；便秘（160）；成人腹泻（192），儿童腹泻（195）；神经系统症状（418）；便血（510）；直肠问题（513）；疑似中风（576）；大便异常（573）。

提醒：记录呼叫者对建议的反应、家庭护理指导以及何时回电。

## 评估与行动

A. 下列情况之一是否出现？

●发作后、昏厥或失去意识后的尿失禁和大便失禁

●突然丧失排便控制力，伴有口齿不清、肌肉无力、视力模糊或双视、意识水平下降

是 “呼叫救护车”

否 ↓ 转到 B

B. 下列情况之一是否出现？

●近期背部受伤、创伤或跌倒后出现剧烈疼痛和多次失禁

●黑便或有血块的大便

是 “立即寻求紧急医疗帮助”

否 ↓ 转到 C

C. 下列情况之一是否出现？

●近期有分娩史，阴道或直肠手术，痔疮，肛瘘或肛裂

●腹痛

●多次失禁发作

是 “24小时内就医”

否 ↓ 转到 D

D. 下列情况之一是否出现？

●感觉肛门内有肿块

●突然发作的腹泻且无法及时上厕所

●近期无排便或多次硬便后频繁不自主渗漏大便

●曾经如厕训练成功的儿童再次出现大便失禁

是 “如果无改善，回电或联系初级保健医生预约”并遵循家庭护理指导

否 遵循家庭护理指导


## 家庭护理指导

大便失禁

●避免便秘。增加膳食纤维、体积和水分摄入（新鲜水果、蔬菜、全谷物、谷类食品和糙米）。尝试使用美达菲来增加体积。每天喝6到8杯水。定期锻炼。

●对于曾经接受过如厕训练的儿童：

●允许孩子决定上厕所的时间。不要强迫孩子坐在马桶上。

●当孩子使用卫生间时给予表扬。

●与孩子讨论全天保持干净可获得的奖励。

●如果孩子弄脏了，让孩子自己清理并更换衣物（如果年龄足够大）。

●不要因为意外而责备或惩罚孩子，也不允许其他兄弟姐妹取笑孩子。

●对于无法及时上厕所的老年或残疾成人，不要表现出愤怒。讨论识别预警信号的方法，并在出现这些信号时尽快去洗手间。

●考虑租赁或购买便携式马桶，以应对突发性腹泻发作时使用。

●审查饮食和用药情况，如使用泻药、新药物、李子汁或蓖麻油的情况。
