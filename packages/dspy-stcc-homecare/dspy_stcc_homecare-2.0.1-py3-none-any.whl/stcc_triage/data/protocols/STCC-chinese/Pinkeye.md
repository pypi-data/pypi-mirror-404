# 红眼病

关键问题：姓名、年龄、发病时间、病因、眼睛是否发红或粉红、历史、用药情况、疼痛评分

其他需要考虑的方案：隐形眼镜问题（163）；眼部受伤（225）；眼科问题（228）；异物入眼（269）；视力问题（639）。

护士提示：如果之前被诊断为红眼病，或有已知或疑似接触过红眼病患者，并且眼睛发红或粉红，请使用本方案。红眼病具有高度传染性。

提醒：记录来电者对建议的反应、家庭护理指导以及何时回电的信息。

## 家庭护理指导

红眼病

● 经常用温水冲洗眼睛，醒着时每1到2小时一次。使用柔软的温暖湿润布料清除结痂和分泌物。

● 如果接触化学刺激物，请用温水冲洗眼睛5分钟。

● 每隔2小时交替敷热敷和冷敷10分钟，持续24小时。

● 不要与家庭其他成员共用毛巾或床单。

● 为了控制瘙痒，可以尝试使用苯那君（Benadryl）24至48小时。遵循标签上的说明。

● 鼓励孩子不要触摸眼睛，并经常洗手。



## 如果出现以下问题，请报告给您的医生/诊所/急诊

● 剧烈疼痛

● 闪光感

● 持续眨眼、流泪或疼痛

● 持续眼部分泌物
