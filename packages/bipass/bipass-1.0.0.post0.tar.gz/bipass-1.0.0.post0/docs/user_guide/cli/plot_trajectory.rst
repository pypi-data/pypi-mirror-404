plot-trajectory
===============

Visualize trajectory data.

.. click:: PASS.apps.plot_trajectory:plot_trajectory
   :prog: pass plot-trajectory
   :nested: full

Description
-----------

This command creates visualizations of trajectory data, showing the balloon's
path over time. Useful for:

- Validating trajectory input files
- Understanding flight patterns
- Correlating position with power system behavior

Example
-------

.. code-block:: bash

   pass plot-trajectory my_config.toml

Output
------

Generates trajectory visualization plots.
