# Тестовый документ для проверки очистки

Это сложный параграф текста с **жирным** и *курсивным* форматированием. Здесь есть точки, запятые, двоеточие: и точка с запятой; также вопросительный знак? И восклицательный! Кавычки "двойные", 'одинарные' и «ёлочки». Скобки (круглые), [квадратные] и {фигурные}. Тире — длинное, дефис-короткий. Проценты 100%, доллары $50, апостроф it's, слеш a/b, обратный слеш c\d. Символы: @email, #хэштег, &амперсанд, *звёздочка, +плюс, =равно, ~тильда, `бэктик`. Числа: 123.45, -67.89, 1,000. Многоточие... и другие знаки препинания!
